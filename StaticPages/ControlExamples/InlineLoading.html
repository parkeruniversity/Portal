Inline Loading Demo

This button will pretend to fire off a process that takes a few seconds.

This button will pretend to fire off a process that takes a fraction of a second.

Here's the code:

$('#normalLoadingButton').click(function () {

	var loadingAnimation = $(this).inlineLoading();
	loadingAnimation.start();

	setTimeout(function () {
		loadingAnimation.stop();

		alert('ALL DONE');
	}, 4000);
});
