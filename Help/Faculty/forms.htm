Forms Overview

The Student Administrator can create and configure a Public Registration form and a Student Survey form.  These forms are created, edited, and deleted by clicking the Setup link in Public Registration and Student Survey portlets.

To create a new form, click the add icon () and proceed to the Form Setup Checklist.

To edit an existing form, click the edit icon () and proceed to the Form Setup Checklist.

To delete an existing form, click the delete icon ()
