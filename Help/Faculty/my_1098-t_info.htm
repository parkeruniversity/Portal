My 1098-T Info

The My 1098-T Info portlet allows a student or their parents to view 1098-T information that has been accumulated for a given calendar year in preparation for claiming tuition paid as a tax deduction.  The portlet allows the user to view preliminary Financial Aid, charges and posted transactions.

Preferences

  • There are no preferences available for this portlet.

 

Settings

  • There are no settings available for this portlet.

 

Security

  • There are no security operations for this portlet.

 

Functions

  • Generate 1098-T Summary Information

    The 1098-T Tuition Statement Data screen displays the student's 1098-T Name, 1098-T Address and information for each required box on the 1098-T form.  A link to View Transactions will display to the right of each piece of data, which will take you to the 1098-T Tuition Statement Box screen.

    To generate the 1098-T Tuition Statement Data:

    • Enter the My 1098-T Info portlet.

    • Click the Go to My 1098-T Info link provided.

    • By default, only 1098-T information for the current year will be provided.  If activated by the JICS Administrator, 1098-T information for previous years will be available in a drop-down list box on the screen.  See Getting Started With Student for more information on how to permit viewing of previous years.

  • Printer Friendly Version

    • Open the My 1098-T Info portlet.

    • Click the Printer Friendly link or the () icon.

    • A new browser window opens and displays the printer friendly version.

    • Click the Send to Printer button.

    • To exit, click the Exit Print Mode link or the () icon.
