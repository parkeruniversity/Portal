Course Authorization Course List

The Course Authorization portlet and this screen are available only if you have permission to do course authorizations.

The Course Authorization List screen displays the courses that you teach and other courses for which you have permissions to do course authorizations.  You can filter the list by selecting a specific term and also by selecting specific divisions and/or departments.

The Setup and Set Options menus are available on this screen.

This screen displays the following:

  • Explanatory text

    The following text is displayed:  "Students may be authorized to register for a course even if the course is full or if the student does not meet a requisite or has a schedule conflict. Click on a course to view authorizations or to add authorizations for that course."

  • Term

    A value in this field is required.  When you select a new term, rows based on the values in the Show box are retrieved.

  • Show

    The values in these fields determine rows that are retrieved:

    • Courses I am teaching

      This radio button is the default when the screen opens.

    • All courses I can authorize

      When you select this radio button, you can select values in the Division and Department drop-down list box to restrict the list of courses retrieved.

    • Division/Program

      This drop-down list box is populated with the divisions/programs in your organization.  An All option is displayed at the top of the list.

    • Department

      This drop-down list box is populated with the departments in your organization.  An All Departments option is displayed at the top of the list and is the default value.

     

When a term is selected, the Show criteria are set, and the Search button is clicked, course rows are retrieved and displayed in a Courses Available for Authorization table at the bottom of the screen.  The information retrieved includes the following:

  • Course; each course is a link to the Course Authorization Details screen

  • Division

  • Requisites and count if the Allow Requisite Authorization check box is selected in the Setup menu; this column will not display if this check box is not selected

  • Capacity and count if the Allow Capacity Authorization check box is selected in the Setup menu; this column will not display if this check box is not selected

  • Schedule Conflict and count if the Allow Schedule Conflict Authorization check box is selected in the Setup menu; this column will not display if this check box is not selected

 

Functions

  • View Authorizations

    • Click any Course link.

    • The Course Authorization Details screen for that course displays the following information:

      • Course Information

      • Course Authorization Totals

      • Show

      • Authorizations Granted (Student, ID, Condition, Reason, Date Added, and Added By)

  • Add Authorizations

    • Click any Course link.

    • The Course Authorization Details screen for that course displays the following information:

      • Course Information

      • Course Authorization Totals

      • Show

      • Authorizations Granted (Student, ID, Condition, Reason, Date Added, and Added By)

    • If you wish, use one of the Show buttons to filter the list.

    • Click the Add New Authorizations button.

    • The Add New Authorizations screen displays.

    • To retrieve the student, enter the ID or the Name in the Add Authorization section and click the Search button.  You can also use Advanced Name Search.

    • If only one student is retrieved:

      • Select the Add check box beside the appropriate Authorization Condition(s).

      • Select the appropriate Reason(s).

      • Click the Add Authorization(s) button.

    • If more than one student is retrieved:

      • The Student Search Results screen displays.

      • Click the Select button beside the appropriate student name.

      • The Course Authorization screen displays with the selected student's name displayed in the Selected Name column.

      • Click the Add Row button for this student.

      • The Add New Authorizations screen displays with the student's name and the Authorization Conditions for which you have permission to add.

      • Select the Add check box beside the appropriate Authorization Condition(s) and select the appropriate Reason.

      • Click the Add Authorizations button.

    • The Course Authorization Details screen displays a message verifying that the authorization has been added.

  • Remove Authorizations

    • Click any Course link.

    • The Course Authorization Details screen for that course displays the following information:

      • Course Information

      • Course Authorization Totals

      • Show

      • Authorizations Granted (Student, ID, Condition, Reason, Date Added, and Added By)

    • If you wish, use one of the Show buttons to filter the list.

    • Select the check box beside the student(s) for which you want to remove authorizations.

    • Click the Remove Authorizations button.

    • A message notifies you that the student has or has not been removed.  If the student is not removed, a reason for not revoking the authorization displays.
