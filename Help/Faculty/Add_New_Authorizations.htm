Add New Authorizations

This screen is only available if you have been given permission to do course authorizations.

The Add New Authorizations screen opens when you click the Add New Authorizations link.  The Conditions (or registration errors) for which a student can be granted authorizations or overrides are the following:

  • Capacity (class size reached)

  • Requisites (not met)

  • Schedule Conflicts

 

Functions

  • Add New Authorization

    From the Add New Authorizations screen:

    • In the Add Authorization box, search for the student by entering their name or ID in the appropriate fields and clicking the Search button.

      or

      Click the Advanced Search link to open the Advanced Name Search screen.

    • You can change your Name Search preferences by clicking the Set Options menu link.

    • If the Name Search returns only one name, the name displays along with the conditions for which you have permission to grant authorization.

    • Select the Add check box beside the Condition.

    • Select the appropriate Reason from the drop-down list.

    • Click the Add Authorization(s) button.

    • If the Name Search returns more than one name, the names display on the Student Search Results screen.

    • Click the Select button beside the student for which you want to add an authorization.

    • The student name will display in the Selected Name column of the Course Authorizations screen.

    • Click the Add Row for this Student button.

    • Select the Add check box beside the Condition.

    • Select the appropriate Reason from the drop-down list.

    • Click the Add Authorization(s) button.
