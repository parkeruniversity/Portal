Degree Audit Detail

The Degree Audit Detail screen displays the details for the requirement click on by the user on the Degree Audit Summary screen.  The user can view the requirements that need to be met to complete their selected program as well as requirements that have already been completed.  The Degree Audit Detail has three sections, as follows:

  • Course Requirements

    The Course Requirements section displays information for the requirement, as follows:

    • Req: This column displays the Requirement Code of the requirement, as determined by the ERP.

    • Description: This column shows is the long description for the requirement, as determined by the ERP.

    • Status: This column will show whether the requirement is met () or not met ().

    • Course/Course Title: For courses that have been taken and successfully complete the requirement, these columns will display the course code and course title.  If the requirement has not been met, it will display a link to the Course Schedules portlet for all classes that meet the requirement.

    • Hours Needed: This column will display the total number of hours needed for the requirement.

    • Hours Earned: If hours contributing to the requirement have been earned, this column will show those hours.

    • Grade Needed: If a specific grade is needed to meet the requirement, it will display in this column.

    • Grade Earned: This column will display the grade earned in the class that meets the requirement.

  • Category Requirements

    The Category Requirements section displays information on any category (group) requirement, as follows:

    • Requirement: This shows the name of the category requirement, with a link that will display the elements of that category requirement.

    • Status: This column will show whether the group requirement is met () or not met ().

    • Needed: The credit hours and/or number of requirements for the group requirement will display in this column.

    • Earned: If any part of the group requirement has been earned, it will display in this column.

  • Other Requirements

    The Other Requirements section displays information for requirements not listed as a Category or Course Requirement, as follows:

    • Requirement: This column displays the name of the requirement.

    • Status: This column displays whether the requirement is Met, In Progress or Not Met.

    • Needed: This column displays what is needed for completion of the requirement.

    • Earned: Those elements of the requirement that have been earned display in this column.

Functions

  • Printing a Degree Audit Report

    • From the Degree Audit Detail screen, click the Printable Degree Audit (PDF File) link.

    • A PDF report of the Degree Audit Summary screen will generate and display in a new window.
