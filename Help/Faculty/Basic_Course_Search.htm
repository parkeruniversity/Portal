Course Search

The Course Search screen allows the user to search for a class by Term, Department, Title, Course Code, Program, Time of Class, Days of Class, Faculty, Campus, Section Status, Hours, Building, or Course Numbers.  When searching by Course Code or Title, the user can search for text that Begins With, Ends With, Contains or is an Exact Match to the text entered.

Results of the search are displayed on the Course Search Results screen.
