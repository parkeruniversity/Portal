Advisee Roster

The Advisee Roster allows you (the advisor) to search for and view a list of your advisees and to view information about a selected advisee.  Information about the advisee is available via links to the following Student portlets/screens:

NOTE:  If any portlet link is not displayed, you have not been given the Allow Access global permission to that portlet by your JICS Administrator.

  • Add/Drop Courses

  • Advisor Meetings (displays meetings for a selected student only)

  • Course History

  • Course Needs

  • Course Schedule

  • Degree Audit

  • Financial Aid Awards

  • Financial Aid Document Tracking

  • GPA Projection

  • Grade Report

  • Major Exploration/What If

  • Residence info (displays only default student life session information for selected student)

  • Student Schedule

  • Unofficial Transcript

The default view allows you to enter search criteria and/or click the Advanced Search link to enter additional search criteria.  When you enter search criteria and click the Search button, the results list is displayed on pages of 25 names per page.  If more than 25 names are returned on the results list, the first 25 names are displayed at the bottom of the search criteria and an alphabet navigation link based on the start and end of additional pages will display.

Example  

Ab - De

De - Mo

Mo - Sa

Sa - Ze

 

The following information displays about each advisee:

Column

Description

*FERPA Restrict

Displays an if a FERPA restriction exists in the Name Master table in EX; this means that this student does not want personal/academic information disseminated

*ID

Displays the advisee's id

*Name

Displays the advisee's name; if the icon is displayed, you can click the icon to view the My Info popup for the advisee

*Needs to Register?

Displays Y if the advisee has not yet registered

*Registration Clearance

Column displays only if the Use the Registration Clearance Process for Selected Students check box has been selected in Course Schedules; button values are Remove and Grant; text values are Not Applicable and Term Expired

NOTE:  This column will not display on the Printer Friendly version if you selected a status of All Advisees, Active Advisees, or Inactive Advisees on the search screens.

Preferences

  • Default Advisee Division

    • From the Advisee Roster portlet, click the wrench icon ().

    • Click the Preferences tab.

    • Select the division you want displayed for the search criteria in the Advisee Roster portlet when the portlet is initially opened.  The options in this drop-down list are defined in the institution's database.

    • Click the Save button.

  • Default Advisee Status

    • From the Advisee Roster portlet, click the wrench icon ().

    • Click the Preferences tab.

    • Select the status you want displayed for the search criteria in the Advisee Roster portlet when the portlet is initially opened.  The following options are available for your selection:

      • All Advisees

      • Active Advisees

      • Currently Registered

      • Inactive Advisees

      • Need Advisor Approval

      • Need Registration Clearance

      • New Students

      • Not Yet Registered

    • Click the Save button.

 

Settings

These settings can be changed by any one who has permission to access the Can Admin operation.  These settings and permissions will affect only this specific instance (copy) of this portlet.

  • Default Student Life Term

    • From the Advisee Roster portlet, click the wrench icon ().

    • Click the Settings tab

    • Select the Student Life Term to be used when viewing an advisee's residence information through the Advisee Roster portlet.

    • Click the Save button.

 

Functions

  • View Advisee Details

    • From the Advisee Roster portlet, enter search criteria and search for advisees.

    • Advisees meeting the search criteria will display below the search criteria.

    • Click the name link of the appropriate advisee.

    • The Advisee Details screen opens and displays detailed information about the advisee and links to other screens/portlets.

     

  • Export to Excel

    • From the Advisee Roster portlet, enter search criteria and search for advisees.

    • Advisees meeting the search criteria will display below the search criteria.

    • Click the Export to Excel link.

    • Excel will open displaying the advisee data.

    • You can manipulate the data as you would in any Excel spreadsheet.

     

  • Email Listed Advisees

    • From the Advisee Roster portlet, enter search criteria and search for advisees.

    • Advisees meeting the search criteria will display below the search criteria.

    • Click the Email Listed Advisees link.  A message notifies you if email addresses of some or all advisees are not available.

    • An email form will open, listing all the available advisee email addresses in the To: column.  

    NOTE:  There is currently no option available for sending an email to selected advisees.  However, you may delete email addresses from the To: column, which allows you to email only certain advisees.

     

  • Printer Friendly Version

    • From the Advisee Roster portlet, enter search criteria and search for advisees.

    • Advisees meeting the search criteria will display below the search criteria.

    • Click the Printer Friendly link or the icon.

    • The application displays the printer friendly version of the advisee list meeting the selected search criteria.

    • Click the Send to Printer button.

    • To exit, click the Exit Print Mode link or the icon.
