Grading Course List

This page displays a list of all your courses for the selected division, year, and term.   However, the courses for the current year/term are displayed by default.

The following fields are available:

Field

Required

Explanation

Instructions Text

 

This default text explains how to enter/change grades.

Division

Yes

This drop-down list is populated with divisions retrieved from your institutional database and allows you to select a division to be included in the search criteria.  All Divisions is also included as an option.  The default may be the last selected division you used.

Term

Yes

This drop-down list is populated with terms retrieved from your institutional database and allows you to select a term to be included in the search criteria.  The default may be the last selected year/term you used.

Search button

 

This button retrieves a list of courses from your institutional database for the selected Division and Term.

 

When the Search button is clicked, the following columns are displayed in the Course List table:

Column

Description

Course

The Course hyperlink opens the Update Student Grades screen (an editable screen) or the Current Student Grade List (a read-only screen) or displays a message informing you that you do not have access to view grades for the course, depending on the security you have been given.

Catalog (CX only)

If allowed, the catalog course code is retrieved from your institution's catalog and displayed in this column.

Title

The title of this course is retrieved from your institutional database and displayed in this column.

Cr Hrs

The number of credit hours for this course is retrieved from your institutional database and displayed in this column.

Clock Hrs

The number of clock hours for this course is retrieved from your institutional database and displayed in this column.

Enter/Change Grades - Midterm

If allowed by your institution, this column displays data that is retrieved from your institutional database.  This column displays whether or not Midterm Grades are editable for a course.

Enter/Change Grades - Final

This column displays whether or not Final Grades are editable.  Final Grades will be editable if the final grading period is open on your institutional database.

 

 

Functions

  • Enter/Change grades

    From the Grading Course List page:

    • Select a Division from the drop-down list.

    • Select the appropriate Term from the drop-down list.

    • Click the Search button to display the list of courses.

    • Click the Course Title to open the Update Student Grades screen.

    • Enter/Update grades.

    • Click the Save button.
