Course Details

The Course Details screen displays information about the selected course and provides links to additional information.  Information on this screen includes name (course code), term, division, credit hours, clock hours, instructor, meeting days, department, location, and status (O=open, C=closed (number of open seats)/(total number of seats)), note (if any), and course description.  

Any Crosslisted Courses are displayed at the bottom of the screen with type, title, capacity, enrollment, and waitlisted data.  Links are provided to the Course Requisites screen, the Course Textbooks screen (if the Administrator has set up JICS to display textbook information), the Catalog Information page (if the Administrator has set up JICS to display catalog information) and a link that displays all courses in that department.

Functions

  • View/Print Course Textbooks

    From the Course Details screen:

    • Click the Course Textbooks link.

    • The Course Textbooks screen displays textbook information for the selected course.

    • Click the Printer Friendly link or the printer icon ().

    • A Printer Friendly Version of the textbook list displays.

    • Click the Send to Printer button.

    • After printing, click the Exit Print Mode link or the exit icon () to return to the Course Textbooks screen.

  • View/Print All Courses in the XXX Department

    • Click the All courses link in the XXX Department link.

    • The Results screen displays a list of all courses in the department to which the previously selected course belongs.

    • Click the Printer Friendly link or the printer icon ().

    • A Printer Friendly Version of the Results screen opens.

    • Click the Send to Printer button.

    • After printing, click the Exit Print Mode link or the exit icon () to return to the Results screen.
