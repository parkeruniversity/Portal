Grade Report

The Grade Report portlet allows students to display and print their grades for a selected Year and Term (a single Session).  The user can also print a custom Grade Report if the portlet is configured to do so.

Preferences

  • There are no preferences available for this portlet.

 

Settings

  • There are no settings available for this portlet.

 

Security

  • There are no security operations for this portlet.

 

Functions

  • View Final Grade Report

    The Final Grade Report shows a simple report detailing each Course Code, Course Title, Final Grade, whether it was a Repeat class, Attempted Credits, Earned Credits, GPA Credits, Quality Points and GPA for a given year and term (a single session).  To access the Final Grade Report:

    • Enter the Grade Report portlet and select the Session (year and term) to view from the drop-down list box provided.

    • Click the View Final Grade Report link to generate a Grade Report for the session selected.

    • On this screen, you can change the Session as needed and generate a .pdf of the Grade Report.  If midterm grades are available, there will be a link to View Midterm Grade Report.

  • View Midterm Grade Report

    CX clients:  Mid-term grades are displayed by view date in the academic calendar record.  Midterm grades are available to students after the date in the view date field of the academic calendar record.

    The Midterm Grade Report (if made available by the JICS Administrator) shows a simple report detailing each Course Code, Course Title, Final Grade, whether it was a Repeat class, Attempted Credits, Earned Credits, GPA Credits, Quality Points and GPA for a given year and term (a single session).  To access the Final Grade Report:

    • Enter the Grade Report portlet and select the Session (year and term) to view from the drop-down list box provided.

    • Click the View Midterm Grade Report link to generate a Grade Report for the session selected.

    • On this screen, you can change the Session as needed and generate a .pdf of the Grade Report.  If midterm grades are available, there will be a link to View Midterm Grade Report.

    Note: If the JICS Administrator has not activated midterm grades, this link will not be available.  Please see Getting Started with Student for further information on how to activate midterm grades for this portlet.

  • Printer Friendly Version

    • Open the Grade Report portlet.

    • Click the Printer Friendly link or the () icon.

    • A new browser window opens and displays the printer friendly version.

    • Click the Send to Printer button.

    • To exit, click the Exit Print Mode link or the () icon.
