Step 1: Define Form

This screen is used by the Student Administrator to establish the essential design features of the form.

  • Use the Form Name text box to change the name of the form, if necessary.

  • If you select the "Yes" radio button for Allow User to Save an Incomplete Form, the user will be able to save their work, even if incomplete.  If you select "No," they will not be able to save until they have fully completed the form.

  • Enter the date in the Activation Date text box or use the calendar icon () to select an activation date.  Please note that you must enter an activate date.

  • Enter the date in the Expiration Date text box or use the calendar icon () to select an expiration date.  You must enter an expiration date for your form.

  • When you are finished with this step, select the "Yes" radio button for Setup Complete.  If you select "No," this option will not show as complete.

  • Click Save to save your form definitions and return to the Form Setup Checklist.
