Advisor Meetings

Advisor Meetings allows students to search for and view scheduled meetings with their advisors.  To search for an advisor meeting during a specific time period, enter the start date and end date into the fields provided or click the calendar icon to select a start and end date.  If no start or end date is provided, the portlet will show all scheduled advisor meetings for the user.

Advisor Meetings also provides advisors with the ability to view, create and update meeting schedules for all of their advisees.  The advisor can search for scheduled meetings as described above.  If the user is logged in as an advisor, Add and Delete buttons will also be visible, which advisors can use to manage meetings with their advisees.

Functions

  • Display Meetings

    • From the Advisor Meetings portlet, enter the date range you wish to search (or leave both date fields blank to view all meetings) and click the Display Meetings button.

    • A list of meetings will display, including the name of the advisee, the date and time of the meeting, the description of the meeting and the status of the meeting.

    • Click the Details link to view the Meeting Details screen for that meeting.

  • Add a Meeting

    • From the Advisor Meetings portlet, click the Add button.  The Add button is only available to individuals logged in as a role with the Is Advisor flag checked in the Global Operations of the portlet.  See Getting Started with Student for information on how to set the Is Advisor Flag.

    • The Meeting Details screen opens and displays default information about the new meeting, which can be edited as appropriate by the user.

  • Delete a Meeting

    • From the Advisor Meetings portlet, enter the date range you wish to search (or leave both date fields blank to view all meetings) and click the Display Meetings button.

    • The list of Scheduled Meetings appears.  Select the meeting(s) by clicking the check box to the left of the advisee name(s).

    • Click the Delete Selected button at the bottom of the screen to delete the selected meeting(s).
