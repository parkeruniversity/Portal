Advisee Details

The Advisee Details screen opens when you click the name of an advisee in the Advisee Roster screen and displays details about the advisee.

The Advisee Details for section displays the following information:

  • Academic Summary

    The following academic data is displayed only if the data is available for the advisee:

    Classification

    Academic Status

    Enrolled Date

    Degree

    Planned Grad(uation Date)

    First Major (will be displayed as Major if Second Major is not defined for the student)

    Max(imum) Credits

    Second Major

    Social Security Benefits (not available for EX)

    First Minor (will be displayed as Minor if Second Minor is not defined for the student)

    Veteran's Benefits (not available for EX)

    Second Minor

    Advisor(s)

     

     

    This information is the same information displayed on the Academic tab of the My Info portlet for the advisee.

  • Holds

    The Holds table displays a single row of information for each hold.  If more than a single row of information is available for the hold, click the + sign to expand the row.  Click the - sign to collapse the row.

    The row of information can contain the Hold Code, Type (registration, transcript, or grades) - Level of Severity (advisory or absolute), Begin Date (not available for EX), Expiration Date (not available for EX), Office (not available for EX), Description, and Remarks (not available for EX).

    NOTE:  This section is not displayed if there are no Holds for the advisee.

    CX clients:  Absolute holds are honored during registration.  They are defined in the configuration table (config_table).  The WEB_REGIST_HOLD_ACTION prevents the student and faculty from registering a student that has an absolute hold.

    This information is the same information displayed on the Academic tab of the My Info portlet for the advisee.
