Course Facilities Search

The Course Facilities Search window allows you to enter the following search criteria:

Criteria

Required

Description

Term

Yes

The drop-down lists available year/term/subterm combinations (in Jenzabar CX, Year/Session; in Jenzabar EX, Year/Term)

Campus

 

The drop-down lists available campuses

Building

 

The drop-down lists available buildings; the default is All Buildings

  

The Retrieve Rooms button retrieves rooms for the selected building (you must select a building) and populates the Room drop-down list.

The Room radio button allows you to select a specific room or the All Rooms (default) criterion from the drop-down and retrieves and displays courses scheduled in that room(s) for the selected building(s) when you click the Search button.  If you change the Building value, the Room drop-down list clears until you click the Retrieve Rooms button again.

The String Search radio button allows you to select the following match:

  • Contains  - The search would retrieve all room names that include the text you enter in the text box.

  • Exact Match  - The search would retrieve all room names that exactly match the text you enter in the text box.

  • Begins With - The search would retrieve all room names that begin with the text you enter in the text box.

  • Ends With - The search would retrieve all room names that end with the text you enter in the text box.

 

Functions

  • Set Default Campus

    • From the Course Facilities Search window, click the Set Options menu link.

    • In the Default Campus drop-down list, select the campus you would like to use as your default when you are searching course facilities.

    • Click the Save button.

    • This default value will display in the Campus drop-down as the default when you perform a course facilities search.

  • Search Course Facilities

    • From the Facility Schedules portlet, click the Course Facilities Search link.

    • Select the appropriate year/term/subterm combination in the Term drop-down list.  This is a required value.  If you do not enter a value, the following message displays:  "You must select a Term to view facility schedules."

    • Select the appropriate location in the Campus drop-down list.

    • Select the appropriate building or All Buildings criterion in the Building drop-down list.

    • Click the Retrieve Rooms button.

    • In the Room Search section, click either the Room or the String Search radio button.

      • If you click the Room radio button, select the appropriate room or All Rooms criterion from the Room drop-down list.  This list was populated when you clicked the Retrieve Rooms button for the particular building(s).

      • If you click the String Search radio button, select your desired match from the drop-down list and then enter the text to match in the text box.

    • Click the Search button.

    • The Search Results screen opens displaying the results.  If no results are found, the following message displays:  "No courses were found for the criteria you selected."
