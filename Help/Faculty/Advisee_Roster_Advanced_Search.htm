Advisee Roster Advanced Search

The Advisee Roster Advanced Name Search screen provides you with additional search criteria fields to use for searching for advisees in your database.  All search criteria is used in a where clause with 'and'.

The following fields/check boxes are available:

Section

Field/Check Box

 Description

Advanced Search Criteria

Advisee Status

Select a status from one of the following values:  All Advisees, Active Advisees, Currently Registered, Inactive Advisees, Need Advisor Approval, Need Registration Clearance, New Students, and Not Yet Registered.  The default value can be specified by clicking the wrench icon on the screen and setting the Default Advisee Status preference.

 

ID

When you enter an ID Number, the Name Search will look for an exact match.

 

Division

Select a division from one of the values in the drop-down list.  Valid division values are defined in the institutional database.  The default value can be specified by clicking the wrench icon on the screen and setting the Default Advisee Division preference.

 

Term

Select a term from one of the values in the drop-down list.  Valid term values are defined in the institutional database.

This field is not available if you selected one of the following statuses:  All Advisees, Active Advisees, or Inactive Advisees.  If you select one of the statuses, all terms will be searched.

 

Degree

Select a degree from one of the values in the drop-down list.  Valid degree values are defined in the institutional database.  

 

Major

Select a major from one of the values in the drop-down list.  Valid major values are defined in the institutional database.  

 

City

You can enter the entire city name or a portion of the city name.

 

State

Select the state from the drop-down list.

 

Last Name

You can enter the entire last name or a portion of the last name.  Name Search will return names determined by the value you select in the drop-down list box (Contains, Exact Match, Begins With).

 

First Name

You can enter the entire first name or a portion of the first name.  Name Search will return names determined by the value you select in the drop-down list box (Contains, Exact Match, Begins With).

 

Middle Name

You can enter the entire middle name or a portion of the middle name.  Name Search will return names determined by the value you select in the drop-down list box (Contains, Exact Match, Begins With).

Name Search Options

 

At least one of the Primary or Secondary Name Source options must be selected.  If not, you will receive the following message:  "You must check 'Use Primary Name Source', 'Use Alternate Name Source', or both options in order to search on a name."

 

Use Phonetic Match

If you select this check box, Name Search will attempt to find a name using a phonetic or Soundex match.

 

Use Primary Name Source

If you select this check box, Name Search will attempt to find a name using the LAST_NAME, FIRST_NAME, and or MIDDLE_NAME fields in the ID record.

 

Use Alternate Name Source

If you select this check box, Name Search will attempt to find a name using the NICKNAME, PREFERRED_NAME, and/or BIRTH_NAME fields in the ID record.

 

 

Functions

  • Searching for Advisee(s)

    From the Advisee Roster screen:

    • Click the Advanced Search link.

    • The Advanced Search screen opens.

    • Enter or select desired search criteria.

    • If you are entering name search criteria, select the Use Primary Name Source check box, the Use Alternate Name Source check box, or both check boxes.

    • Click the Save button.

    • The Advisee Roster screen opens and displays the results of the search.
