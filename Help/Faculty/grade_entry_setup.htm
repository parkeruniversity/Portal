Grade Entry Setup

The Setup menu link displays on the Grade Entry portlet if you have permission to administer global settings.       

 

Global Settings

There are three global settings:

  • Use Midterms

    This setting allows you to hide or display midterm grade information.  If you select the check box, midterm grade information is displayed.  If you clear the check box, midterm grade information is hidden.

  • Allow Absence Entry

    This setting determines whether faculty have the ability to enter number of absences.  If you select the check box, faculty can enter the number of absences for students.  If you clear the check box, faculty will be unable to enter the number of absences for students.

  • Allow Clock Hours Entry

    This setting determines whether faculty have the ability to enter the number of clock hours student attend class.  If you select the check box, faculty can enter the number of clock hours for students.  If you clear the check box, faculty will be unable to enter the number of clock hours for students.
