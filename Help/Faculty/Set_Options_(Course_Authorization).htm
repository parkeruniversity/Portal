Set Options (Course Authorization)

The Set Options screen allows you to change your personal options and indicate the options you would like to use when doing a Name Search.  You can also indicate which column in the Authorizations Granted table you would like to use for sorting when the Course Authorization Details screen opens.

The Setup menu is also available from this screen.

 

Preferences

  • Name Search Options

    You can select one of three options to use when doing a Name Search:

    • Select the Search Using Phonetic Match check box  if you want to have the Name Search attempt to find a name using a phonetic or Soundex match.

    • Select the Search Using the Primary Name check box if you want the Name Search attempt to find a name using the current name in the ID record.

    • Select the Search using the Alternate Name Type check box if you want the Name Search attempt to find a name using names other than the main name record.  Other names could include types such as formal, informal, matric, nickname, preferred name, former name, etc.

  • Default Authorizations Sort Order

    The Sort Column drop-down lists three columns:  Student, Condition, and Reason.  Select the column on which you want the Authorizations Granted table to be sorted when the Course Authorization Details screen opens or when you select a different course.
