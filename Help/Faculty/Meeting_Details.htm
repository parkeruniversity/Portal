Meeting Details

The Meeting Details screen displays information concerning the specific meeting selected.  If the user is logged in as an advisor, the fields on this screen will be editable.

The following fields are available:

  • Name: If the user is logged in as a student, this field will display the name of the student's advisor.  If the user is logged in as an advisor, this field is a drop-down list box showing all advisees assigned to that advisor.  If a new meeting is being created, an advisee must be selected before the meeting can be saved.

  • Date/Time: If the user is logged in as a student, this displays the date and time of the meeting, as determined by the advisor.  If the user is logged in as an advisor, these four fields will be editable.  In the date field, you can enter the date (month/day/year) into the text box or click on the Calendar icon () to select a date.  The time fields are drown-down menus that allow you to select the time for the meeting.

  • Description: If the user is logged in as a student, this displays the advising code of the meeting.  If the user is logged in as an advisor, this field will be a drop-down list box showing the advising codes available, as determined in the ERP.

  • Status: If the user is logged in as a student, this field displays the status of the meeting.  If the user is logged in as an advisor, this field will be a drop-down list box showing the available meeting status codes as determined in the ERP.

  • Purpose: If the user is logged in as a student, this field displays the purpose of the meeting, as determined by the student's advisor.  If the user is logged in as an advisor, this will be an editable text box.

  • Notes: If the user is logged in as a student and the portlet is configured to allow students to view Meeting Notes ,  this field displays notes for the meeting, as determined by the student's advisor (see Getting Started with Student for more information on configuring this portlet).  If the user is logged in as an advisor, this will be an editable text box.

The following operation is available to those logged in as a student:

  • Request New Meeting with Advisor: Clicking this link will open your local email software with the advisor's email address entered in the To: field.

The following operations are available to those logged in as advisors:

  • Send Meeting Notification To Advisee: Clicking this link will open your local email software with the advisee's email address entered in the To: field.

  • Save: Clicking this button will save the meeting and return the user to the Advisor Meetings window.

  • Save and Schedule: Clicking this button will save the meeting and reset the Meeting Details window to its default values so that the user can enter a new meeting.

  • Reset: Clicking this button will reset the Meeting Details screen to its original values.

  • Cancel: Clicking this button will return the user to the Advisor Meetings window without saving any changes to the meeting.
