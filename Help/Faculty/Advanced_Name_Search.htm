Advanced Name Search

The Advanced Name Search window allows the user to find a student based on several criteria, as follows:

  • Name Search Options

    In this section, the user can choose to conduct the search for an ID number using  by Using Phonetic Match, Primary Name and Alternate Name Types.  Select the appropriate check boxes for each option that you wish to use.

     

  • Name Search Fields

    This section contains several fields that you can use to find the Student you are searching for.  You can choose to search using Any of Criteria or All of Criteria using the radio buttons provided.  For criteria, you can enter the ID or full or partial information for City, State, Last Name, First Name and/or Middle Name.  Use the drop down list box for each criteria to determine if it Contains the information entered, or if it is an Exact Match, Begins With what is entered or Ends With the information.

    When all information is entered, click the Search for Name button.

  • Student Search Results

    The Student Search Results screen will display the results of your search, including information on the student's name, ID, Major, Classification, Current Division and Location, if available.  This information only displays if the student has elected to be available for selection as a roommate preference.

    Press the Select button to return to the Roommate Preferences screen with the ID number of the student select already entered in the text box on that screen.
