File Cabinet

Your File Cabinet is a portlet located within your My Pages context (on a page also called File Cabinet). Only you have access to your File Cabinet.

You can use your File Cabinet to store any of the following:

·       Bookmarks

·       Coursework

·       Course Cartridges

·       Handouts

·       Readings

After you store materials in your File Cabinet, you can import items from the File Cabinet into any of the portlets mentioned in the bulleted list above. With this feature, you can re-use materials as often as needed.
