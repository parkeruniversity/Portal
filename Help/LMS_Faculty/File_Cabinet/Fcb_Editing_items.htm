Editing items

After you have added items to your File Cabinet, you may want to edit or delete them. Expand any of the following to see details on changes you can make to these items from within the File Cabinet.

Here is an overview of the types of changes you can make:

·       You can change any attributes of bookmarks, readings, and handouts.

·       You can change the name of any assignment. You can also search for individual questions and delete them from a saved assignment.

·       You cannot make changes to course cartridges.

For more details, expand any of the following:

+ To make changes to any bookmark, handout, or reading

1.        Within the File Cabinet, select the appropriate tab.

2.        Locate the item that you want to modify. Click the corresponding pencil icon.

3.        Make any changes that are appropriate.

4.        Click Save.

+ To change the name of any assignment

1.        Within the File Cabinet, select the Coursework tab.

2.        Locate the item that you want to rename. Click the corresponding pencil icon.

3.        Make the appropriate changes.

4.        Click Save.

+ To delete a question from a saved assignment

1.        Within the File Cabinet, select the Coursework tab.

2.        Click the Search for Questions link.

3.        Use the form displayed to find the questions(s) you want to delete:

a.        Enter the appropriate keyword(s).

b.        If desired, use the drop-down list to narrow your search to one particular question type.

c.        If desired, switch from the default choice of Search questions only to Search questions and answers.

d.        Click Search.

4.        In the search results displayed, use the checkboxes at the left to choose the questions to delete.

5.        At the bottom of the screen, click Delete Selected.

+ To delete any item in the File Cabinet

1.        Within the File Cabinet, select the appropriate tab.

2.        Do one of the following:

o      If you want to delete only one item, select the corresponding trash-barrel icon.

o      If you want to delete multiple items, use the checkboxes at the left to select them, then click Delete Selected.

3.        When the system asks whether your are sure you want to continue, click OK.
