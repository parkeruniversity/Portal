Adding items

There are a few ways to add items to the File Cabinet.

+ Saving from another portlet

For most types of content, you can create the item in another portlet -- such as Handouts or Bookmarks -- and then save it to the File Cabinet. In each of the following portlets, you should see an option labeled "Save to the File Cabinet" or something similar.

You can create content, and save it to the File Cabinet, from the following portlets:

·       Bookmarks

·       Coursework

·       Handouts

·       Readings

In addition, you can import items from outside the portlet, directly into the File Cabinet, from the Course Content Import portlet.

+ Creating items within the File Cabinet

You can use the File Cabinet to create and store the following types of items.

Type of content

How to add it from within the File Cabinet

Bookmarks

Navigate to the Bookmarks tab and click Add a Bookmark.

Handouts

Navigate to the Handouts tab and click Upload Files to File Cabinet.

Readings

Navigate to the Readings tab and click Add a New Reading.

You cannot create coursework or upload course cartridges from within the File Cabinet.
