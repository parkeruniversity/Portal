Working with folders

In any of the File Cabinet tabs, you can create folders for organizing the items saved in that tab. Note that there is only one level of folder organization. In other words, you cannot nest folders within folders.

+ To create a folder

1.        Navigate to the File Cabinet, as described in Navigating to the File Cabinet, and select whichever tab you want to create a folder for.

2.        Click the link labeled Create a new folder to display the File Cabinet - Add a Folder screen.

3.        Enter a name for the folder.

4.        Click Save.

+ To rename a folder

1.        Navigate to the File Cabinet, as described in Navigating to the File Cabinet, and select whichever tab you want to create a folder for.

2.        Locate the folder you want to rename and click the corresponding Edit icon, which looks like a folder with a pencil on it.

3.        Modify the name as appropriate.

4.        Click Save.

+ To delete a folder

1.        Navigate to the File Cabinet, as described in Navigating to the File Cabinet, and select whichever tab you want to create a folder for.

2.        Using the checkboxes at the left, select the folder(s) you want to delete.

3.        Click Delete Selected.

+ To move items to a folder

1.        Navigate to the File Cabinet, as described in Navigating to the File Cabinet, and select whichever tab you want to create a folder for.

2.        Locate the item(s) that you want to move. If appropriate, click the name of a folder to display its items.

3.        Using the checkboxes at the left, select the item(s) you want to move.

4.        At the bottom of the screen, locate the drop-down list labeled Move Selected to:

5.        Using this drop-down list, select one of the following:

o      The folder to which you want to move these items.

o      The Up one level choice, which moves the item out of a folder and to the top level of the tab. (This option is displayed only if you are working within a folder.)

6.        Click Go.
