Setting permissions for a sub-section

If you have created a sub-section, and you want members of the Students role or another role to be able to see and use it, you must give them permission to view at least one of the pages within the sub-section.

Note that even if you did not manually create a page for the sub-section, by default it contains at least one page (which was created by the system when the sub-section itself was created).

1.        To set permissions for a sub-section:

a.        In the sidebar, expand the My Courses label.

b.        Click the name of the course for which you want to create a sub-section.

2.        In the sidebar, click Context Manager to display the Context Manager screen, with the Properties tab selected by default.

3.        In the sidebar, click on the name of the sub-section for which you want to set permissions.

4.        Navigate to a page that you want students (or other users) to have access to. Click the Edit page link (which is at the top right of the page). The system displays a series of tabs that let you customize this page.

5.        Click the Permissions tab.

6.        Use the checkboxes at the right to select the roles that should be allowed to view this page.

7.        Click Save.
