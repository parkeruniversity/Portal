Adding a page

At times you may want to add a page to your course context.

Similarly, if you have created a new sub-section, you may want to add a page to it. By default, a newly created sub-section has only one page. If you want to add more pages to the sub-section, use this procedure.

To add a page to a sub-section:

1.        Navigate to the appropriate course context.

2.        If appropriate, click on the name of the sub-section to which you want to add a page.

3.        In the sidebar, click the Add a Page link. The system displays the Create a new page screen.

4.        In the Name field, enter a name for the page you are creating.

5.        Click Create. The system creates the page.
