Creating a sub-section for a course

A sub-section is a essentially a collection of pages within a course context. The link to a sub-section is represented as a folder icon in the sidebar (although sub-section icons can be hidden).

You might create a sub-section in the following situations:

·       If you have a number of pages on a single topic that you think should be grouped for the convenience of students, or for your own convenience.

·       If you want to make certain content available only to certain roles.

+ To create a sub-section for a course

1.        Display a course context.

a.        In the sidebar, expand the My Courses label.

b.        Click the name of the course for which you want to create a sub-section.

2.        In the sidebar, click Context Manager to display the Context Manager screen, with the Properties tab selected by default.

3.        Click the Sub-Sections tab to display a screen listing any sub-sections that already exist.

4.        Click Create a New Sub-Section to display the Create a New Sub-Section screen.

5.        In the Name field, type a name. This name will be displayed in the sidebar, if the course context is configured to display sub-section icons in the sidebar. It will also be displayed in breadcrumbs. For these reasons, you should choose a meaningful name. Note that the name will also become part of the URL, and once the sub-section is created, you will not be able to change the URL (even if you change the name).

6.        Click Create.
