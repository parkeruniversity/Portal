Setup

The Setup screen allows your institutions to indicate whether they wish to allow faculty users to create and manage course authorizations for the override of various error conditions at the time registration occurs.

 

The Set Options menu is also available from this screen.

 

Global Settings

  • Use Course Authorization Functionality in CRM Faculty

    If you select this check box to allow faculty or other staff to use the authorization functionality, then you should indicate which of the following authorization options you wish to include in your Course Authorization processing:

    • Allow Section Requirement Authorizations

      Select this check box if you want to allow faculty or other staff to authorize that selected students can register for a course in spite of the section requirement error at the time registration occurs.

      If the Use Course Authorization Functionality in CRM Faculty is selected and the Allow Section Requirement Authorizations check box is selected, then Section Requirements will appear as an option in the Conditions drop-down if the faculty/staff has rights to maintain section requirements and the column (what column?) will appear on the default screen.

    • Allow Schedule Conflict Authorizations

      Select this check box if you want to allow faculty or other staff to authorize that selected students can register for a course in spite of the schedule conflict error at the time registration occurs.

      If the Use Course Authorization Functionality in CRM Faculty is selected and the Allow Schedule Conflict Authorization check box is selected, then Schedule Conflicts will appear as an option in the Conditions drop-down if the faculty/staff has rights to maintain schedule conflicts and the column (what column?) will appear on the default screen.

    • Allow Requisite Authorizations

      Select this check box if you want to allow faculty or other staff to authorize that selected students can register for a course in spite of the requisite error at the time registration occurs.

      If the Use Course Authorization Functionality in CRM Faculty is selected and the Allow Requisite Authorization check box is selected, then Requisites will appear as an option in the Conditions drop-down if the faculty/staff has rights to maintain requisites and the column (what column?) will appear on the default screen.

    • Allow Capacity Authorizations

      Select this check box if you want to allow faculty or other staff to authorize that selected students can register for a course in spite of the Course Capacity error at the time registration occurs.

      If the Use Course Authorization Functionality in CRM Faculty is selected and the Allow Capacity Authorization check box is selected, then Capacity will appear as an option in the Conditions drop-down if the faculty/staff has rights to maintain capacity and the column (what column?) will appear on the default screen.
