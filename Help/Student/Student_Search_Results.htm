Student Search Results

The Student Search Results screen displays the results of Name Search when you click the Search for Name button if more than one name matches the search criteria.

If only one name matches the search criteria, the Course Authorization screen opens and displays the student name in the Selected Name column.

If more than one name matches the search criteria and if you want to select one of the names, click the Select button beside the appropriate student name, which opens the Course Authorization screen and displays the student name in the Selected Name column.  If you do not want to select one of the names, click the Cancel button.

The following information is displayed on this screen:

Column

 Description

Student

Displays the name of the student matching criteria during Name Search

ID

Displays the student ID

Major

Displays the student’s major

Classification

Displays the student’s class year

Current Division

 Displays the student’s division

Location

Displays the student’s location

Campus

Displays the student’s campus
