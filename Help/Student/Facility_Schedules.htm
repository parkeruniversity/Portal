Facility Schedules

The Facility Schedules portlet allows you to search for and retrieve a list of rooms and view courses scheduled in those rooms.

Set Options

When you click this link, the Set Options screen displays and allows you to select a default Campus to use when you use the Course Facilities portlet.

  • Set Default Campus

    • From the Course Facilities Search window, click the Set Options link.

    • In the Default Campus drop-down list, select the campus you would like to use as your default when you are searching course facilities.

    • Click the Save button.

    • This default value will display in the Campus drop-down as the default when you perform a course facilities search.

Functions

  • Course Facilities Search

    The Course Facilities Search link on the Default screen opens the main Course Facilities Search window where you can enter search criteria and retrieve and view courses assigned to specific rooms and buildings.
