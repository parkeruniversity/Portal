Financial Aid Administration

The Financial Aid Administration portlet allows financial aid personnel to see financial aid awards or missing/received documents for a selected student.  This portlet calls screens from the Course Authorizations, Financial Aid Awards and Financial Aid Document Tracking portlets.  

The user can either enter in a student ID, or click on the Advanced Name Search link, which opens the Student Name Search screen from the Course Authorization portlet.  If the user enters a student ID on the default screen or selects a student from the results list, they then are shown a screen from which they have the option to open the Financial Aid Awards portlet or the Financial Aid Document Tracking portlet to see missing/received documents.

Preferences

  • There are no preferences available for this portlet.

 

Settings

  • Click the wrench () from the Financial Aid Administration portlet and click the Settings tab.

  • Enter the subject of the alert in the Missing Document Alert Message Subject text box.  This will be the subject of the email sent to a student to notify them of a missing financial aid document.  The alerts functionality is only available if the Financial Aid Document Tracking portlet is accessed through the Financial Aid Administration portlet and the Can Send Alerts permission is granted to the logged-in user.

  • Enter the text of the alert in the Missing Document Alert Message Template text box.   This will be in the message body of the email sent to the student to notify them of a missing financial aid document.  The document name and award year may be dynamically included in the body of the alert if the @@Doc and/or the @@YEAR variables are specified in the template text.  The alerts functionality is only available if the Financial Aid Document Tracking portlet is accessed through the Financial Aid Administration portlet and the Can Send Alerts permission is granted to the logged-in user.

  • Click the Save button.

 

Security

  • Click the wrench () from the Financial Aid Administration portlet and click the Security tab.

  • Select the check boxes beside the particular roles you want to be able to send alerts for missing Financial Aid documents in the Can Send Alerts column.  This causes the Alerts check box to be displayed next to each item listed in the Missing Documents section of the Financial Aid Document Tracking portlet for users of the selected roles.  The alerts functionality is only available if the Financial Aid Document Tracking portlet is accessed through the Financial Aid Administration portlet and the Can Send Alerts permission is granted to the logged-in user.

  • Click the Save button.

 

Functions

  • View Financial Aid Awards

    To view Financial Aid Awards:

    • Enter the Financial Aid Administration portlet

    • Enter the ID of the student in the text box provided or use the Advanced Search link to the Advanced Name Search screen.

    • If multiple students meet the criteria, you will be taken to the Student Search Results screen.  Click the Select button to select a student.

    • On the Financial Aid Details screen, click the link for View Financial Aid Awards to go to the Financial Aid Awards screen.

  • View Financial Aid Documents

    To view Financial Aid Documents:

    • Enter the Financial Aid Administration portlet

    • Enter the ID of the student in the text box provided or use the Advanced Search link to the Advanced Name Search screen.

    • If multiple students meet the criteria, you will be taken to the Student Search Results screen.  Click the Select button to select a student.

    • On the Financial Aid Details screen, click the link for View Financial Aid Documents to go to the Financial Aid Document Tracking screen.

  • Printer Friendly Version

    • Open the Financial Aid Administration portlet.

    • On any screen, click the Printer Friendly link or the () icon.

    • A new browser window opens and displays the printer friendly version.

    • Click the Send to Printer button.

    • To exit, click the Exit Print Mode link or the () icon.
