Textbook Search

The Textbook Search screen allows you to search for existing textbooks to assign to the selected course.  After you enter search criteria and click the Search button, the results display at the bottom or the screen.

The following fields are available on this screen:

Field

Description

Course

Displays selected course code

Term

Displays Year/Term

Course Title

Displays title of selected course

Division

Displays selected Division

Book Title

Allows you to enter a book title to be used as a search criterion

Author

Allows you to enter the name of an author to be used as a search criterion

ISBN

Allows you to enter an ISBN to be used as a search criterion

Publisher

Allows you to enter a publisher name to be used as a search criterion

 

The Textbook table may display the following information for books returned by the search:

Field

Description

Select

Allows you to select one or more books to associate with the selected course

Title

Displays title of the textbook

Author(s)

Displays names of the textbook author(s)

Price

Displays the amount your institution pays for the textbook

Fee

Displays the amount the student will be charged for the textbook

ISBN

Displays the assigned International Standard Book Number (ISBN)

Publisher

Displays the name of the textbook publisher

Description

Displays a description of the textbook

 

 

Functions

  • Search for Textbook

    From the Faculty Course Control portlet:

    • Click the Textbook Entry option in the Go Directly To drop-down.

    • The Textbook Entry screen displays.

    • Click the Search Existing Textbooks to Assign to Course link.

    • The Textbook Search screen opens.

    • Enter desired criteria in the Book Title, Author, Publisher, and/or ISBN fields.

    • Click the Search button.

    • If textbooks that match the search criteria are found in the Textbook Definition table, they will display at the bottom of the screen.

    • If you want to assign a textbook(s), select the check box beside the appropriate textbook(s).

    • Click the Assign Selected Textbook(s) button.

    • The Textbook Entry screen opens and displays the new textbook added.

  • Add New Textbook to Assign to Course

    From the Faculty Course Control portlet:

    • Click the Textbook Entry option in the Go Directly To drop-down.

    • The Textbook Entry screen displays.

    • Click the Add New Textbook To Assign To Course link.

    • The Add a Textbook screen opens and displays the selected course code, course title, term, and division.

    • Enter required and optional information.

    • Click the Save and Add Another button if you are adding more than one textbook.

    • Click the Save button when you have entered all new textbooks.

    • The Textbook Entry screen opens and displays textbooks added.
