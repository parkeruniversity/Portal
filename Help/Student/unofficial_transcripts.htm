Unofficial Transcripts

The Unofficial Transcripts portlet displays GPA, session, cumulative statistics, and detailed course information by program and term, as for an official transcript.

The user can choose to view their unofficial transcript by Program in the drop-down list box provided.

Preferences

  • There are no preferences available for this portlet.

 

Settings

  • There are no settings available for this portlet.

 

Security

  • There are no security operations for this portlet.

 

Functions

  • Generate PDF Report

    If your Administrator has this option available, there will be a link at the bottom of the window to generate a PDF version of your unofficial transcript.  Clicking this link will automatically generate a PDF file that you can save to your desktop or print for later use.

    See Getting Started With Student for information on how to set up this link.

  • Printer Friendly Version

    • Open the Unofficial Transcripts portlet.

    • Click the Printer Friendly link or the () icon.

    • A new browser window opens and displays the printer friendly version.

    • Click the Send to Printer button.

    • To exit, click the Exit Print Mode link or the () icon.
