Textbook Entry

The Textbook Entry screen opens when you click the Textbooks option in the Go Directly To drop-down.  The screen displays the list of textbooks for this course.  If you do not have permission to enter or edit textbooks for this course, the following message displays:  "You do not have access to Textbook Entry for this course."

The Course drop-down includes courses you are associated with for the selected Term and Division.  You can view textbooks for a different course by selecting that course in the Course drop-down list.  However, if you need to change the Term and Division, you must make that change on the Faculty Course Control screen.

The Search Existing Textbooks to Assign to Course link opens the Textbook Search screen.  The Add New Textbook to Assign to Course link opens the Add a Textbook screen.  Clicking the Edit icon () opens the Edit a Textbook screen.

In addition to the course code, the following information is displayed for textbooks:

Field

Description

Functions

  • Add New Textbook to Assign to Course

    From the Faculty Course Control portlet:

    • In the Go Directly To drop-down, click Textbooks to open the Textbook Entry screen for a specific course.

    • Click the Add New Textbook to Assign to Course link.  This link is also available on the Textbook Search screen.

    • The Add a Textbook screen opens.

    • Enter required (Book Title) and other desired optional information.

    • Click the Save button or the Save and Add Another button if you are adding more than one textbook.

    • The Textbook Entry screen displays the textbook(s) added.

  • Search Existing Textbooks to Assign to Course

    From the Faculty Course Control portlet:

    • In the Go Directly To drop-down for the desired course, click Textbooks to open the Textbook Entry screen.

    • Click the Search Existing Textbooks to Assign to Course link.

    • The Textbook Search screen opens.

    • You may enter Book Title, Author, Publisher, and/or ISBN to be used as search criteria.

    • If textbooks that match the search criteria are found in the Textbook Definition table, they are listed in the Textbooks table at the bottom of the screen.

    • Select the check box(es) of those you want to add to the course.

    • Click the Add Assigned Textbook(s) button.

    • The Textbook Entry screen displays the textbook(s) assigned.

  • Edit a Textbook Associated with this Course

    From the Faculty Course Control portlet:

    • In the Go Directly To drop-down for a particular course, click Textbooks to open the Textbook Entry screen.

    • Click the Edit icon () beside the textbook you want to edit.

    • The Edit a Textbook screen opens.

    • Edit information as necessary.  All fields are editable.

    • Click the Save button.

    • The Textbook Entry screen displays the edited textbook information.

  • Remove a Textbook Associated with this Course

    From the Faculty Course Control portlet:

    • In the Go Directly To drop-down, click Textbooks to open the Textbook Entry screen for a specific course.

    • Click the Delete icon () beside the textbook you want to delete.

    • A prompt screen displays the following message:  "Are you sure you want to delete this item?"

    • Click the OK button to delete the textbook.

    • The textbook is deleted from the Textbook Entry screen.
