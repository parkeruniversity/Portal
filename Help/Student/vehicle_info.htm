Vehicle Info

The Vehicle Information portlet displays information on the Student's Vechicle Description, Assigned Parking Location, Type of Permit, Permit Number, Permit Expiration Date, License Plate and notes any Violations for that Student.

If any violations are listed as Outstanding, a link will be provided to display a Violation Description, Violation Date, and Violation Fee.

Preferences

  • There are no preferences available for this portlet.

  • There are no settings available for this portlet.

 

Security

  • There are no security operations for this portlet.

 

Functions

  • Printer Friendly Version

    • Open the Vehicle Info portlet.

    • Click the Printer Friendly link or the () icon.

    • A new browser window opens and displays the printer friendly version.

    • Click the Send to Printer button.

    • To exit, click the Exit Print Mode link or the () icon.
