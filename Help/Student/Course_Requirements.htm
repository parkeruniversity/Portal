Course Requirements

The Course Requirements screen displays detailed information on the requirements needed and accomplished for a specific aim or sub-audit.  This screen displays only when the student has more than one program requirement.  Each Requirement is listed, along with a Description of the requirement, what is Needed to fulfill the requirement (credits, a specified GPA, etc.) and what has already been Earned.

If students or advisors are allowed to recalculate student progress (as determined in the Setup for the portlet), a Recalculate Student Progress button will appear on the screen.  Click this button to pull updated information from the ERP to show up-to-date information on the requirements needed.
