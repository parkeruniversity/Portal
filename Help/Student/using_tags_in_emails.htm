Using Tags in Emails

Tags are used to insert personal candidate information into emails you create.  If you use a tag in an email template, be sure it is also an assigned question on the inquiry or application you created.  

The following example shows the use of tag insertion:

  • From the Email Template screen, compose your email in the Email Message text box

  • If you desire to use any of the available tags, type @@ and the appropriate tag in the location you want it to appear in the email

  • Upon sending the email, the system retrieves the information from your database and replaces the tags with the desired data
