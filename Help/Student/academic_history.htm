Academic History

The Academic History screen opens when you click the Academic History link on the Advisee Details screen for a selected student.

NOTE:  This screen and the Academic tab on the My Info portlet for the student displays the same information.

The Education table displays a row for each degree received (if any).  The row contains the name of the institution where the degree was conferred, degree, major, GPA, graduation date, and transcript.  If no information exists for the student, the Education table will not display.

The Exams table displays a row for each candidate exam (SAT, ACT, etc.) the advisee has taken.  The row contains the name of the test, the date the test was taken, composite score, and each individual element score.  If no test data exists for the student, the Exams table will not display.
