Major Exploration and What If Scenarios

Major Exploration and What If Scenarios allows students to view requirements for selected courses, and if configured, to apply completed courses towards any of the programs the institutions offers.

This portlet uses some of the screens from the Course Needs portlet to provide students with information on what is required to complete the selected program.  All required course and group requirements that have not been satisfied will be displayed, indicating how much credit needs to be earned to complete the requirement and the current status of the requirement.  If there is a category or group requirement, the user will be able to view those elements that are required to complete the requirement.  For each course displayed the user will be able to view any current course offerings that would satisfy the requirement.

Functions

  • Generate Major Exploration Scenario

    The Major Exploration screen displays detailed information on the requirements needed and accomplished for a specific aim or sub-audit.  Each Requirement is listed, along with information specific to that requirement.

    If students or advisors are allowed to recalculate student progress (as determined in the Setup for the portlet), a Recalculate Student Progress button will appear on the screen.  Click this button to pull updated information from the ERP to show up-to-date information on the requirements needed.

    To generate a Major Exploration Scenario:

    • Open the Major Exploration portlet

    • Select a Major/Minor from the drop-down box provided.

    • If you wish to apply the student's current class history, check the Apply My Courses checkbox.

    • Click the View button.  This will generate the Major Exploration Scenario information.

    The following screen will show the Course Requirements needed for the Scenario, as follows:

    • Req: This column displays the Requirement Code of the requirement, as determined by the ERP.

    • Description: This column shows is the long description for the requirement, as determined by the ERP.

    • Status: This column will show whether the requirement is met () or not met ().

    • Course/Course Title: For courses that have been taken and successfully complete the requirement, these columns will display the course code and course title.  If the requirement has not been met, it will display a link to the Course Schedules portlet for all classes that meet the requirement.

    • Hours Needed: This column will display the total number of hours needed for the requirement.

    • Hours Earned: If hours contributing to the requirement have been earned, this column will show those hours.

    • Grade Needed: If a specific grade is needed to meet the requirement, it will display in this column.

    • Grade Earned: This column will display the grade earned in the class that meets the requirement.
