Form Setup Checklist

To create a new form and enter the Form Setup Checklist:

  • Click on the Setup link in the Public Registration or Student Survey portlets to view a list of all available forms assigned to that portlet.  

  • Click on the plus icon to the right of the Forms heading.

  • Enter the name of the form in the Create a form named textbox.  

  • If there is another form you would like to use as a model, select it from the Use this previously created form as the model drop down list box.  If there is no previously created form that you can use as a model, leave this drop down list box blank.

  • Click Save to proceed or Cancel to exit.

  • The Form Setup Checklist will appear with a series of options (hyperlinks) that allow the Student Administrator to access form configuration screens.  Those options with an X ( ) have not yet been completed.  Those with a check () have been completed successfully.
