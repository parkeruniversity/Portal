Unsuccessful Save

This screen displays the following text if there was an unsuccessful save of your grade changes:  "Grades were not successfully saved to the database."  If available, an error message explaining the reason why the save was unsuccessful will also display.

Two links are available:

  • The Return to Previous Page link returns you to the Update Student Grades screen, where the grades you entered will still be displayed.

  • The Return to Course List link takes you to the Grading Course List page.
