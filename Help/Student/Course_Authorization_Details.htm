Course Authorization Details

You will be able to view authorizations for courses that you have rights to view and /or maintain.

The Course Authorization Details screen opens when you click on a course link.  The screen displays data for students for whom authorizations/registration overrides have been granted under the Authorizations Granted heading.  The screen allows you to view, add, and remove authorizations, provided you have been given rights to do so.  The Setup and Set Options menu options may also be available.

If you have permissions to manage Course Authorizations, you will be able to select courses based on your rights.  If you do not have permissions to manage Course Authorizations, you will be able to view information only for the courses you teach.

If you have permissions to manage Course Authorizations, you will have access to the Remove Authorizations button and the Add New Authorizations link.

The following fields/columns are displayed on the screen:

Field/Column

Description

Course Header

Displays course code, term description, division description, catalog description, number of credit hours, capacity limit, number enrolled, and the number waitlisted

Course Authorization Totals

Displays the total number of authorizations for the course for each Condition that is displayed on the screen

Show

Since All Conditions is the default, all authorizations are retrieved and displayed when you open the Course Authorization Details screen. However, you can filter the conditions you want to view by selecting one of the following radio buttons:

The Capacity radio button (when clicked) retrieves and displays authorizations for the Capacity Condition.

The Requisites radio button (when clicked) retrieves and displays authorizations for the Requisites Condition.

The Schedule Conflict radio button (when clicked) retrieves and displays authorizations for the Schedule Conflict Condition.

The Show buttons will display for each condition displayed on the screen.

Authorizations Granted

Displays the most recent student row for each condition for which authorization was granted; displays student name, ID Number, condition, reason, date added, and person who added the authorization; this data is from the Course Authorizations table in EX

 

 

Functions

  • Add New Authorizations

    From the Course Authorization Details screen:

    • Click the Add New Authorizations link.

    • The Add New Authorizations screen opens for this course.

    • In the Add Authorization box, search for the student by entering their name or ID in the appropriate fields and clicking the Search button.

      or

      Click the Advanced Search link to open the Advanced Name Search screen.

    • You can change your Name Search preferences by clicking the Set Options menu link.

    • If the Name Search returns only one name, the name displays along with the conditions for which you have permission to grant authorization.

    • Select the Add check box beside the Condition.

    • Select the appropriate Reason from the drop-down list.

    • Click the Add Authorization(s) button.

    • If the Name Search returns more than one name, the names display on the Student Search Results screen.

    • Click the Select button beside the student for which you want to add an authorization.

    • The student name will display in the Selected Name column of the Course Authorizations screen.

    • Click the Add Row for this Student button.

    • Select the Add check box beside the Condition.

    • Select the appropriate Reason from the drop-down list.

    • Click the Add Authorization(s) button.

    • The Course Authorizations Detail screen re-opens and displays the added authorization(s).

  • Remove Authorizations

    From the Course Authorization Details screen:

    • If you wish, use one of the Show buttons to filter the list.

    • Select the check box beside the student(s) for which you want to remove authorizations.

    • Click the Remove Authorizations button.

    • This removes the row from the screen and adds a row to the Course Authorization table with a status of Revoked (EX, Registration, Task, Course, Courses, Authorizations tab).
