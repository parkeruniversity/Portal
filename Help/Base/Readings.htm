Readings

The Readings portlet lets authorized users post a list of recommended or required readings for students and others. Each reading can be accompanied by a link -- such as a link to an online bookstore or a library page.

Individual readings are posted within sets. For example, a portlet instance might include a set made of readings related to certain topics, certain authors, or certain units in a class.

Typically the default and maximized views of the portlet show selected details about each reading, and you can click the name of any reading to see more details about it, such as its description and the name of the publisher.
