Understanding the Forums views

The default view of the Forums portlet shows a list of recent posts.

When you maximize the portlet, the system displays a larger view, which is the Forum Home. In addition to showing recent posts, this view shows a summary of each category, with a list of each category’s topics. Each category is a collection of topics. Each topic is a collection of posts and replies.

As you click around within the Forum Home, the view will change. To re-display the main Forum Home view at any time, click the name of the portlet instance.
