About My Subscribed Forums

The My Subscribed Forums view exists only if you have subscribed to Forums content using the My Subscribed Forums option. After you do this, the My Subscribed Forums view is accessible from any Forums portlet instance. It shows messages from all Forums areas for which you have chosen the My Subscribed Forums as your subscription option.

+ To display the My Subscribed Forums view

1.        Navigate to the Forum Home of any Forums portlet instance.

2.        Click the link labeled View My Subscribed Forums. If this link is not displayed, then you do not have any subscriptions that can be viewed this way.

+ Using the My Subscribed Forums view

The My Subscribed Forums view is similar to the regular Forums view, except that it shows only content that you have subscribed to, all aggregated in one place.

If desired, you can use this view to filter material from only one portlet instance at a time (if you have subscribed to more than one). To filter, use the My Subscribed Forums drop-down list at the top of the screen.

If you want to display the actual Forums portlet instance that an item came from, you can use the links in the Category | Topic and Forum columns to navigate to the appropriate place.
