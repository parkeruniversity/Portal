Sending e-mail

You have a few options for sending e-mail within the Group Directory portlet.

+ To send e-mail to all users

1.        Navigate to the Group Directory portlet instance and maximize it:

2.        At the bottom of the screen, select the checkbox labeled Select All.

3.        Click Open E-mail to open a form.

4.        Select one of the following:

o      Select Public if it's OK for the recipients to see on another's e-mail addresses. Note that in this case you can also optionally choose to send a copy to yourself.

o      Select Private to hide all of the e-mail addresses. Note that in this case a copy will automatically be sent to you and BCC the other recipients.

5.        Fill out the Subject and Body fields as appropriate.

6.        If you want to add an attachment, click the Browse button to locate the file and populate the File field. Then make sure you click Add to actually attach the file to the message.

7.        Click Send.

 

+ To send e-mail to selected users

1.        Navigate to the Group Directory portlet instance and maximize it.

2.        Use the checkboxes next to each person's name to select the recipients of the e-mail.

3.        At the bottom of the screen, click the link labeled E-mail Selected Members.

4.        In the form displayed, select one of the following:

o      Select Public if it's OK for the recipients to see on another's e-mail addresses. Note that in this case you can also optionally choose to send a copy to yourself.

o      Select Private to hide all of the e-mail addresses. Note that in this case a copy will automatically be sent to you and BCC the other recipients.

5.        Fill out the Subject and Body fields as appropriate.

6.        If you want to add an attachment, click the Browse button to locate the file and populate the File field. Then make sure you click Add to actually attach the file to the message.

7.        Click Send.

 

+ To send e-mail to all members of one role

1.        Navigate to the Group Directory portlet instance and maximize it.

2.        At the bottom of the screen, set the E-mail All drop-down list to the role that you want to e-mail. For example, in the Coursemates portlet, you might choose Students.

3.        Click the Open E-mail button.

4.        In the form displayed, select one of the following:

o      Select Public if it's OK for the recipients to see on another's e-mail addresses. Note that in this case you can also optionally choose to send a copy to yourself.

o      Select Private to hide all of the e-mail addresses. Note that in this case a copy will automatically be sent to you and BCC the other recipients.

5.        Fill out the Subject and Body fields as appropriate.

6.        If you want to add an attachment, click the Browse button to locate the file and populate the File field. Then make sure you click Add to actually attach the file to the message.

7.        Click Send.
