Viewing a list of groups you belong to

If you need to view a list of groups that you currently belong to, use the procedure below.

To view a list of groups that you belong to:

Do one of the following --

·       Start from the sidebar: In the Quick Links area, expand the My Groups menu option. The system displays a list of all the groups you currently belong to. Clicking the name of any group will take you to the context for that group.

·       Use the Campus Groups portlet:

    a.        Navigate to an instance of the Campus Groups portlet that is set to show the My Groups view. (You can tell whether the portlet instance is set up this way if it has a label at the top that says My Groups.) Often, there is a portlet instance set up this way on the My Pages tab.

    b.        Review the list. Note that this list includes both groups that you belong to and ones that you have been invited to join. If you want to visit the context for any particular group, click the name of the group. Note that this view also provide links that let you of drop groups and accept invitations.
