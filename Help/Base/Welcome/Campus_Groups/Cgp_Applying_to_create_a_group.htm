Applying to create a group

Only people with administrative privileges can create a group, but you can apply to create a group if you have access to a Campus Groups portlet set up in the New Group Application view or the My Groups view. When you apply, your application is forwarded to an administrator for review. If it is approved, you group will be created, with you as the Leader.
