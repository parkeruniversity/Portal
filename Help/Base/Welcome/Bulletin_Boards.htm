Bulletin Boards

The Bulletin Boards portlet is a discussion area. Users can post messages and browse messages posted by others. To reply, you can use controls within the portlet to send the author an e-mail. Note that Bulletin Board messages are grouped into topics, which are then grouped into categories. You must navigate to a topic to post and review messages.

These help pages describe how to post messages, reply to messages, and set preferences. If you need help managing an instance of the Bulletin Boards portlet, see Jenzabar's Internet Campus Solution: Administration Guide or e-Racer: Faculty Guide.
