Setting preferences

To set preferences for how content displays when you browse a particular Readings portlet instance, click the wrench icon and select the Preferences tab. You have the choices described below. After you make your selections, be sure to click the Save button at the bottom of the screen.

+ Order Readings Alphabetically

Select this option to order readings alphabetically by their display names. Note that if you select this, the portlet will ignore any sequence that might have been manually set up by the person who manages the portlet instance.

+ Order Sets Alphabetically

Select this option to order sets alphabetically by their display names. Note that if you select this, the portlet will ignore any sequence that might have been manually set up by the person who manages the portlet instance.

+ Show Empty Sets

When you select this checkbox, the portlet will show sets that do not yet contain readings. Otherwise, empty sets are hidden.
