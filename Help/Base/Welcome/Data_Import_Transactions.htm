Data Import Transactions

The Data Import Transactions portlet shows the details about data import from your school's ERP database to the JICS database. This transfer happens through a service called the Data Import Service, or DIS. The Data Import Transactions portlet shows the data file and log information for recent imports.  

The default view of the portlet shows the five most-recent transactions. If you maximize the portlet instance (by clicking its name), the portlet shows the 30 most-recent transactions, including the transaction data and transaction log file.

Note that you run the DIS from the ERP system. For details on doing this, consult your ERP system's documentation.
