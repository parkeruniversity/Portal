Searching for people

If desired, you can search for a particular user.

To search the directory:

1.        Navigate to the directory by click the Campus Directory link in your portal's footer. Doing this opens the directory's Browse tab.

2.        Click the Search tab.

3.        Enter the appropriate details in the First Name and Last Name fields.

4.        Click Search. In response, the portal returns details about this user, if he or she has not hidden their details from the directory. If the person was found, note that you can click the blue information icon next to the person's name to see that user's My Info portlet, may include the user's photo and/or contact information.

If you can't find the Campus Directory link, it may be that the feature has been hidden by your portal administrator.
