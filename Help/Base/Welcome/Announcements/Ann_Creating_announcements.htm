Creating announcements

Note that only certain users have permission to create announcements.

To create an announcement:

1.        Maximize the portlet instance.

2.        Choose Manage > New Announcement. If you do not see the Manage menu, then you do not have permission to create announcements.

3.        In the Title field, enter a short summary for the announcement.

4.        Under Send To, choose the users who will be able to view your new announcement. You can use the checkboxes next to the various roles, and if appropriate you can also click Add Individual Users to select specific people.

5.        In the Details field, enter the text of the announcement.

6.        If appropriate, select start and end dates, and choose whether the announcement should be archived or deleted when the end date has been reached.

7.        Click Save.
