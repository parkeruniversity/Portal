Setting preferences

To set your personal preferences for the Bulletin Boards portlet, click the wrench icon and select the Preferences tab. You have the choices described below. After you make your selections, be sure to click the Save button at the bottom of the screen.

+ Default Screen Topic

If the portlet includes multiple topics, you can use this drop-down list to select the one that you want to display by default when the portlet is not maximized. (When the portlet is maximized, all topics always display.)

+ Results Per Page

You can use this drop-down list to select the maximum number of results that the system will show per page after a search.
