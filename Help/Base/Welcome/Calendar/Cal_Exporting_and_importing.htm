Exporting and importing

If appropriate, you can export items from a Calendar portlet instance into an Excel CSV file. You can export items from any Calendar portlet instance that you have permission to view.

Further, when working in your My Calendar, you have the ability to import calendar items from a CSV file. (Depending on your permissions, you may also be able to import into other Calendar portlet instances.) Note that the portlet instance into which you are importing items must already host at least one calendar.

+ To export calendar items

1.        Navigate to an instance of the Calendar portlet.

2.        Maximize the portlet instance.

3.        Using the checkboxes next to the calendar names, select the calendars whose events you want to export, and de-select the ones whose events you want to hide. (The calendar names are listed under the Calendar and Subscription headings.)

4.        Click Export Events for this view to create and download an Excel file.

 

+ To import calendar items

1.        Navigate to the appropriate Calendar portlet instance, such as your My Calendar.

2.        Maximize it.

3.        Click Manage Calendars.

4.        Click Import Events from an external source to open a form.

5.        Use the Browse button to locate the CSV file you want to import.

6.        Use the Import To drop-down list to choose a calendar to import the items into.

7.        Click Save.
