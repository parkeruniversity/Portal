Creating calendars

If desired, you can create a personal calendar in your My Calendar portlet instance to host your personal events. If you are a faculty member or belong to another group that has additional privileges, you can also create calendars anywhere that you have permission.

+ To create a calendar

1.        Navigate to the appropriate portlet instance and maximize it.

2.        Click Add a Calendar to open a form.

3.        In the Calendar Name field, enter a name.

4.        Depending on your permissions, you may see a checkbox labeled Allow others to subscribe to this Calendar. If desired, select this.

5.        If appropriate, enter text in the Description field.

6.        Click Save.

+ To modify a calendar

1.        Navigate to the appropriate calendar instance and maximize it.

2.        Click the Manage Calendars link. The screen updates to include a section labeled My Calendars.

3.        Locate the calendar you want to modify and click the corresponding pencil icon to open a form.

4.        Make any changes necessary. Note that you may be able to modify any of the following settings:

o      Calendar Name

o      Allow others to subscribe to this Calendar (This is shown only to certain users, depending on their permissions).

o      Description

5.        Click Save.

+ To delete a calendar

1.        Navigate to the appropriate calendar instance and maximize it.

2.        Click the Manage Calendars link. The portlet opens a view with a section labeled My Calendars.

3.        Locate the calendar you want to delete and click the corresponding trash-barrel icon.

4.        When the system presents a pop-up asking whether you're sure, click OK.
