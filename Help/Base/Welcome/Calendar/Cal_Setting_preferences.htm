Setting preferences

To set your personal preferences for the Calendar portlet, click the wrench icon and select the Preferences tab. You have the choices described below. After you make your selections, be sure to click the Save button at the bottom of the screen.

+ Day Click Action

This preference affects the behavior of the portlet if it is maximized and you click on a specific day. You have the following options:

·       Open the daily view -- The portlet will open a view that shows the day in detail.

·       Add an event -- The portlet will open a form that lets you create an event.

Note that this preference will be ignored if you do not have permission to create events, or if you have set the Default Maximized Calendar View preference to Day, Year, or Event List.

Additionally, if the Calendar portlet is not maximized and you click on a specific day, the portlet will always open a detailed view of the day, regardless of your Day Click Action preference.

Even if you don't choose Add an event for this preference, if you have permission to create events you can always do so using the Add an event links that appear throughout the portlet.

+ Default Maximized Calendar View

This setting determines what happens when you maximize the portlet instance. You have the following choices:

·       Day -- The portlet will show a detailed view of the day.

·       Week -- The portlet will show a detailed view of the week.

·       Month -- The portlet will show a detailed view of the month.

·       Year -- The portlet will show a detailed view of the year.

·       Event List  -- The portlet will show a list of upcoming events.

+ Week Start Day

This preferences determines the first day of the week as shown in certain views.

+ Day Start Time

This preference determines the time of day listed at the beginning of the daily view. Events that occur earlier than this time will be shown on the previous day's daily view. For example, if you choose 8 a.m. for your Day Start Time, and an event occurs from 7 a.m. to 7:30, that event will not be present in your daily view for that day. To see it on a daily view, you will have to look at the previous day. Other types of views, such as the week and month views, will still associate the event with the actual calendar day that it occurs on.
