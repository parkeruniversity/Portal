For more information

This portal may include more than one help system. Generally, in addition to this help system -- which describes the base product, which all customers have -- there is also one help system for each optional module that your school may have purchased.

If you are looking for details on a portlet that is not described in this help system, try going back to the portlet you need information on and clicking the help icon in the upper right-hand corner -- this should open the appropriate help pages. Note, however, that certain portlets may not be associated with any help system.

Additional documentation may be available to you in PDF form. Check Jenzabar's support site.

+ To check Jenzabar's support site

1.        Open a browser and navigate to MyJenzabar.net/ics.

2.        Log in to the site.

3.        Choose the Support tab, then use the pane at the left-hand side of the screen to navigate to the appropriate page.
