About the reply options

The portlet lets you set a few preferences related to what information is included in your replies. These affect the default setup of your replies (which you can modify on a case-by-case basis). You set these preferences through the View/Reply Options dialog box. You have the following options:

+ Should the system quote the post to which you are responding?

If desired, you can have the system automatically quote the post (or reply) to which you are responding. If you select this option, the system will add the author's name and the text of their post, inside quotation marks, to your own reply.

If desired, you can delete the quotation on a message-by-message basis. However, if you do not select this option, there is no way to add the quote in the same style that the system would (though you can copy and paste the text, and apply your own formatting).

+ Should the system by default include your photo?

The View/Reply Options dialog box includes on option for specifying whether or not your photo, by default, should be set to display with your reply. Regardless of how you set this, you can choose to add or hide your photo on a message-by-message basis, overriding your general preference, by selecting or de-selecting the Show My Photo option on the screen where you create the message.

If you want to display a photo, note that your privacy settings must be set to allow your photo to be shared. To check this, choose Personal Info > Privacy Settings and look at the Photo checkbox. For your picture to display, the Photo checkbox should be selected.
