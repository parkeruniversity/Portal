Adding a post

To add a post:

1.        Navigate to the appropriate Forums portlet instance.

2.        Open the topic where you want to create a post. While viewing the topic, you should see the Add a Post button. If you do not see the Add a Post button, check to make sure you are actually viewing a topic -- not a category, and not another post. If you do not see Add a Post button, you either do not have permission to submit posts, or the topic has been set up to be read-only. (You can tell a topic is read-only if the topic is marked with an eye icon. To see if it is marked with this icon, navigate to the category level and look at it from that view.)

3.        Click Add a Post to open the New Post form.

4.        Fill out the form as appropriate. If you need help with any particular field, see Attributes of posts and replies. Note that the Subject field is required.

5.        If desired, click Preview. While looking at the preview, do not click Cancel unless you want to discard the post. (If you want to make changes, click Edit.)

6.        Click Submit to publish the post. In response, the system either publishes the post or displays a message stating that moderator approval is required before your post can be published.
