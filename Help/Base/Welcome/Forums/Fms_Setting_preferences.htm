Setting preferences

Use this procedure to set preferences related to:

·       How posts are formatted when you view them (your view options).

·       The information included in your replies (your reply options).

These preferences are specific to whichever instance of the Forums portlet you are currently using -- so if you use multiple portlet instances, you may want to set your view and reply options in each one.

To set the view and reply options:

1.        Log in to the portal. (If you are not logged in, you won't be able to set preferences, even if you have permission to post without being logged in.)

2.        Navigate to the appropriate Forums portlet instance.

3.        Display the full text of any post or reply.

4.        At the top of the screen, click the button labeled View/Reply Options or View Options. (If you do not have permission to reply, you can only set view options.)

5.        In the View/Reply Options dialog box (or View Options dialog box), make the appropriate selections in each category. For help understanding the options, see About the view options and About the reply options.

6.        Click Save Options.
