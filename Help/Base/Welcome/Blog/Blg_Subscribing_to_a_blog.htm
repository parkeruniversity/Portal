Subscribing to a Blog portlet instance

If you want to be able to review posts without navigating to a Blog portlet instance, you should subscribe to the instance. You can subscribe using any of the following methods:

·       'My Subscriptions' -- With this option, posts from the Blog portlet instance are displayed in your My Subscriptions view. (For more details on displaying the My Subscriptions view, see Using the My Subscriptions view.) You can switch to the My Subscriptions view from any Blog portlet instance, or you can create a Blog/My Subscriptions view on your My Pages context.

·       Your e-mail account -- Each blog post is mailed to you with a subject line reading "New post in Name of tab." The message also displays a link that lets you jump to the post as displayed in the Blog portlet instance that hosts it.

·       Your Message Center -- When a Blog item is posted, your Message Center displays an alert with the subject line, "New post in Name of tab." You can click through to read the full text of the message within the Message Center. The message also displays a link that lets you jump to the post as displayed in the Blog portlet instance where the item was posted.

·       RSS feed -- With this option, you can use a third-party reader to display Blog posts. The reader displays the 20 most recent posts. Note that this option is available only if an administrator has configured the portlet instance to allow it.

Use any of the following procedures as appropriate:

+ To start subscribing via My Subscriptions, e-mail, or Message Center

To subscribe via My Subscriptions, e-mail, or Message Center for the first time:

1.        Navigate to the portlet instance and maximize it.

2.        Click the Subscribe link, which is near the bottom of the portlet, to display a screen showing the available subscription options.

3.        Select as many checkboxes as are appropriate.

4.        Do one of the following:

o      If one of the options you are selecting is "My Subscriptions," and if you want to place a Blog/My Subscriptions view in your My Pages context for the first time, click the link labeled Add a Blog portlet to your My Pages. In response, the system automatically saves all the subscription selections you just made and creates a My Subscriptions page, with a Blog portlet set up for viewing subscriptions.

o      If you are not choosing to create the My Subscriptions page for the first time, then click the Subscribe button to save your choices.

Note: If you want to create a My Subscriptions page and you do not see a link that allows you to do this, that probably means that you have not yet set up your My Pages context. To resolve this, navigate to the My Pages context and following the links of the setup wizard.

+ To modify subscription options

1.        Navigate to the portlet instance and maximize it.

2.        Click the Subscribe link, which is near the bottom of the portlet, to display a screen showing the available subscription options.

3.        Modify the selections as appropriate.

4.        Do one of the following:

o      If one of the options you are selecting is "My Subscriptions," and if you want to place a Blog/My Subscriptions view in your My Pages context for the first time, click the link labeled Add a Blog portlet to your My Pages. In response, the system automatically saves all the subscription selections you just made and creates a My Subscriptions page, with a Blog portlet set up for viewing subscriptions.

o      If you are not choosing to create the My Subscriptions page for the first time, then click the Subscribe button to save your choices.

Note: If you want to create a My Subscriptions page and you do not see a link that allows you to do this, that probably means that you have not yet set up your My Pages context. To resolve this, navigate to the My Pages context and following the links of the setup wizard.

+ To subscribe via RSS feed

1.        Navigate to the portlet instance and maximize it.

2.        Click the Subscribe link, which is near the bottom of the portlet, to display a screen showing the available subscription options. If RSS subscriptions are allowed, the page includes a box that states "To Subscribe by RSS, add the following feed to your reader: URL."

3.        Copy the URL displayed.

4.        Navigate to your reader and follow whatever instructions are offered for adding the feed.
