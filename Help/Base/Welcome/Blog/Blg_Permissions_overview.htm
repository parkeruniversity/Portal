Permissions overview

Your ability to use the Blog portlet varies depending on your permissions. Expand the bullet items below to see more about what each group of users can do. Note that permissions are specific to an instance of the portlet -- so you might have broad permissions in one instance, but not in another.

+ All users

Any user with the ability to display a Blog portlet instance can:

·       Subscribe to the portlet instance. With this feature, you can receive notifications about new posts via e-mail, Message Center, or your My Subscriptions view. Depending on the instance's setup, you might also be able to subscribe to an RSS feed. For details, see the Subscribing to a Blog portlet instance topic.

·       Use a Blog portlet instance to review items that were posted there, including any comments posted. For details, see the Reviewing posts in the Blog portlet topic.

+ Users with permission to comment

In some cases, you can comment on a Blog post. Your ability to comment depends on the way the portlet instance was configured, and on the way the author set up the post. This is described in Working with comments.

+ Users with permission to post blog items

When the portlet instance is maximized, it may include a link labeled Add a New Post. In this case, you have the ability to post Blog items. You can also:

·       Edit and delete your own posts.

·       Enable commenting on your posts (subject to restrictions an administrator might have established for the portlet instance).

For details, see Creating, modifying, and deleting posts.

+ Administrators

An administrator can:

·       Specify the maximum size for an individual post.

·       Configure a portlet instance to generate an RSS feed.

·       In any given portlet instance, set default values for options available to authors when they post Blog items.

·       Specify the layout of the portlet instance.

·       Manage permissions.

·       Delete users' comments.

For details on any of these tasks, see Jenzabar's Internet Campus Solution 7.x: Administration Guide.
