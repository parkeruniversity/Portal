Working with comments

Depending on the configuration of the portlet instance and on your own permissions, you might be able to add a comment about a Blog post. For details, review "To post a comment," below. Note that if you have permission to comment, then you also have permission to delete your own comments.

+ To post a comment

1.        Locate the Discussion area of the post on which you want to comment. To do this, display the full text of the post. Scroll down -- the Discussion area is at the bottom of the page. If the post does not contain a Discussion area, that means that the author did not enable commenting for this post. If the Discussion area of the post includes a text box labeled Post a Comment, then you have permission to comment. If this box is not displayed, you don't have permission to comment, and you shouldn't continue with these steps.

2.        Enter your comments in the text box.

3.        Click Save.

 

+ To delete a comment that you posted

1.        If you have not already, locate the Discussion area of the post that contains the comment you want to delete. (To do this, display the full text of the post and scroll down to the bottom of the page, where you should see the Discussion area).

2.        Locate the comment you want to delete. Click the corresponding trash barrel icon.

3.        The system will display a pop-up asking if you're sure you want to delete the post -- click OK.
