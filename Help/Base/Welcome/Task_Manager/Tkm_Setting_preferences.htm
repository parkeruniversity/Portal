Setting preferences

The default setup of the Task Manager is that the Display Tasks Due choice defaults to Within 7 days each time you visit the portlet, which means that the default view shows only items that will be due within a week's time. However, you can change this.

To set Task Manager preferences:

1.        Click the wrench icon and select the Preferences tab.

2.        Under By default, display tasks due, set the drop-down list to one of the following:

o      Within 7 days

o      Within 30 days

o      All

3.        Click Save.
