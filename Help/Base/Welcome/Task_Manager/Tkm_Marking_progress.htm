Marking progress

Periodically you may want to update the status of a task assigned to you.

To update the status of a task:

1.        Navigate to an instance of the Task Manager and maximize it. By default, the tab labeled My To-Do List is selected.

2.        Use the Show drop-down list and the Display Tasks Due option to find the item you want to manage.

3.        Click the name of the appropriate task to open a page of details about it.

4.        Use the Status drop-down list to select any of the following:

o      Not Started

o      In Progress

o      Deferred

o      Waiting

o      Completed

5.        Click Update Status.
