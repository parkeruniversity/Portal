Adding custom info

The Custom Info tab lets you add details about yourself that other users will be able to see in your My Info pop-up. You can add text and then group the text items into different categories. Categories are highlighted with gray bars as headings, and items are listed beneath the categories they belong to.

+ To add a category

1.        Log in to the portal.

2.        Click Personal Info.

3.        Select the Custom Info tab.

4.        Click Add Category.

5.        In the Category field, enter a name for your category. This label will display as a heading in a gray bar (assuming the default portal styles are being used).

6.        In the List this Category field, select the position of this category relative to your other categories.

7.        Click Save.

+ To add an item

1.        Log in to the portal.

2.        Click Personal Info.

3.        Select the Custom Info tab.

4.        Click Add an Item.

5.        In the Name field, enter a name for your item. This will display in bold, followed by a colon, as a label for the rest of the item.

6.        In the Info field, enter a description of the item. This test will display to the right of the Name.

7.        In the Category field, select which category this item should belong to.

8.        In the List This Item field, choose the position of this item relative to other items in the category.

9.        Click Save.

+ To modify a category or item

1.        Log in to the portal.

2.        Click Personal Info.

3.        Select the Custom Info tab.

4.        Locate the category or item you want to modify and click the corresponding pencil icon. The system will open a form.

5.        Make whatever changes are necessary. You can change any of the following:

o      The category or item's name.

o      The placement of the category or item relative to others.

o      If you are editing an item:

§       Its description.

§       The category that it belongs to.

o      If you are editing a category, note that you can also move items out of that category and into another category.

6.        Click Save.

+ To delete a category or item

1.        Log in to the portal.

2.        Click Personal Info.

3.        Select the Custom Info tab.

4.        Locate the category or item you want to delete and click the corresponding trash-barrel icon. (Note that you cannot delete the default category created by the system. This category is named Ungrouped, unless you have changed the name.)

5.        When the system presents a dialog asking if you are sure you want to continue, click OK.
