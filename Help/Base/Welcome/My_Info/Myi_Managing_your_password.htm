Managing your password

Depending on how your school manages passwords, you may be able to make password changes within the portal. Specifically, you may be able to change your password, and in some cases create a password hint (which can then be used to reset a forgotten password).

+ To change your password

1.        Log in to the portal.

2.        Click Personal Info.

3.        Click the Password tab. (If you don't see this tab, then the portal is not set up to allow you to change your password this way. You may need to contact an administrator for guidance on how to proceed.)

4.        Enter your current password in the Old Password field.

5.        Enter your desired new password in both the New Password and Confirm Password fields.

6.        Click Save.

+ To create a password hint

1.        Log in to the portal.

2.        Click Personal Info.

3.        Click the Password tab

4.        Click the link labeled Password Hint. If you don't see this link displayed, then the portal is not set up to allow you to create a hint.

5.        Enter a security question in the field labeled Hint Question.

6.        In the Answer field, enter the question's correct response.

7.        Click Save.
