Posting a Bulletin Board item

Note that only certain users have permission to create bulletin board messages.

To create a bulletin board message:

1.        Maximize the portlet instance.

2.        Open a topic. If you are looking at a list of topics within a category, or if you are looking at a list of categories, you will not be able to post.

3.        Click the link labeled Post a Message. If you do not see this link, then you do not have permission to post messages.

4.        In the Subject field, enter a short summary for the item.

5.        In the Message field, enter any additional details.

6.        If appropriate, click Browse to add an attachment.

7.        If appropriate, use the Delete After drop-down list to specify that the item should be removed at some point.

8.        Select or de-select the Show My Photo option.

9.        If desired, preview the item by clicking Preview. If you want to change the message while you are previewing, click Edit. Do not click Cancel unless you want to discard the message.

10.    Click Post a Message to publish your item.
