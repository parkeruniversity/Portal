Creating bookmarks

Bookmarks are links to other pages in the portal or elsewhere on the web. Authorized users can create bookmarks and group them into sets.

To create a bookmark:

1.        Maximize the portlet instance.

2.        Choose Manage > Add a Bookmark. If you do not see the Manage menu, then you do not have permission to create bookmarks.

3.        If you want the bookmark to display as an actual link then you should skip the Label field. However, if you prefer for the link to be represented by a label -- for example, "Google" instead of  "http://www.google.com," enter the label here.

4.        Enter the link in the URL field.

5.        If appropriate, group the bookmark into a set using the Set drop-down list.

6.        If desired, enter text in Description field. If you enter text here, it will be displayed below the bookmark's label.

7.        If appropriate, select start and end dates, and choose whether the bookmark should be archived or deleted when the end date has been reached.

8.        Click Save.

For more details on creating and managing bookmarks and sets, refer to Jenzabar's Internet Campus Solution: Administration Guide or e-Racer: Faculty Guide.
