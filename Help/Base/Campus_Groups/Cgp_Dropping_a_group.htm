Dropping a group

If you need to drop a group, use this procedure. Note that you are allowed to drop a group only if you are a member or a co-leader. A sole leader cannot drop a group because it cannot be without a leader. If you are in this situation and need to drop, appoint another member to be co-leader using the Manage This Group page for the context.

To drop a group:

1.        Navigate to an instance of the Campus Groups portlet that is set up using the My Groups view.

2.        Locate the name of the group that you want to drop and click the corresponding Drop Group link.
