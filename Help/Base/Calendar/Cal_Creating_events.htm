Creating events

If desired, you can add your own personal events to your My Calendar portlet instance. If you are a faculty member or another user with additional privileges, you may also be able to create events in other Calendar portlet instances.

Before you can create an event, the portlet instance must have at least one calendar that can host your event. (For help with this, see Creating Calendars).

+ To create an event

1.        Navigate to the appropriate portlet instance and maximize it.

2.        Click Add an Event. The system displays a form.

3.        In the Event Name field, enter a name for the event.

4.        If appropriate, enter text in the Description field.

5.        Use the Calendar drop-down list to choose the calendar that will host the event. Note that you can click Add a Calendar to create a new calendar.

6.        If desired, enter details in the Location field.

7.        If appropriate, select All Day Event. Doing this removes the End Date and timestamp fields.

8.        Use the Start Date and End Date fields to specify the exact start and end of your event.

9.        If this is a recurring event, use the Occurs area to specify how the event recurs.

10.    Click Save.

+ To modify an event

1.        Maximize the Calendar portlet instance.

2.        Click the name of the event you want to modify. The screen updates to include additional details.

3.        Click Edit this Event.

4.        On the form displayed, make any changes that are necessary.

5.        Click Save.

 

+ To delete an event

1.        Maximize the Calendar portlet instance.

2.        Click the name of the event you want to delete. The screen updates to include additional details.

3.        Click Delete this Event.

4.        When the system presents a pop-up asking whether you're sure, click OK.
