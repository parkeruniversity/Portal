Customizing your view

Any instance of the Calendar portlet might host one or more calendars. You can display as many or as few of the available calendars as you like.

To customize your view:

1.        Navigate to an instance of the Calendar portlet.

2.        Maximize the portlet instance. The screen will update to include more details. Any calendars associated with the portlet instance will be listed under the Calendars and Subscriptions headings.

3.        Using the checkboxes next to the calendar names, select the calendars whose events want to view, and de-select the ones whose events you want to hide.

See also Setting preferences for more details on customizing your view.
