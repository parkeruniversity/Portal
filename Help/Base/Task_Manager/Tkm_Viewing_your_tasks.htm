Viewing your tasks

You can use any instance of the Task Manager portlet to view tasks currently assigned to you.

To view your tasks:

1.        Navigate to an instance of the Task Manager and maximize it. By default, the tab labeled My To-Do List is selected.

2.        Use the Show drop-down list to select the tasks that you want to view. Note that you can choose to view tasks that were created in other instances of the Task Manager portlet (in addition to the one that you're currently viewing). You can use the Display Tasks Due: links to further refine the list of items shown.
