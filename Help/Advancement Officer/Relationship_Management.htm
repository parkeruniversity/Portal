Relationship Management

The Relationship Management portlet generates profiles of constituents and prospective constituents and allows updates to the database as new information is made available to advancement officers.

To use the Relationship Management portlet, first located and select a constituent via a search. Clicking a hyperlinked name in the search results displays the Relationship Management screen, which has four tabs of information that describe the constituent, as detailed below.

Tabs

  • Constituent Profile

    This tab provides a summary of significant data about the constituent that may include name, ID number, alternate addresses, spouse, relationships, education, employment, involvements, accomplishments, interests, giving history, contacts, news, notes, and holds.

    A company's summary may include contact information, corporate data, matching policies, giving history, and contacts.

  • Moves Management

    This tab may provide contacts, solicitations, status, giving clubs, premiums, funding tendencies, and assigned staff related to the constituent.

  • Giving History

    This tab may provide a summary and details of gifts, annual contribution history, pledge tracking, and planned giving.

  • Initiatives & Grants

    This tab may provide patron, grantmaker, recipient, and related staff information of initiatives and grants associated with the constituent.

    Functions

    • Manage

      The Manage menu includes Submit Updates option, which opens the Edit & Submit Updates to Constituent Information screen.  This screen allows you to review, edit, and/or delete any updates you have entered before the final submission.   You must have the proper security in order to submit updates.  

      • Submit Updates

        Any updates you make must be submitted to the institution database.  The Alumni/Development Office personnel will then review and process the submissions and update the database.

        • From the Manage menu, click Submit Updates.

        • The Submit Updates screen displays a list of updates that have been made.

        • If you want to edit the update(s) before submitting them, click the Edit icon () to open the Edit Updates screen.  You can change the date or edit the text entry.  Click the Save button.

        • If you want to delete update(s), click the associated Delete icon ().  You will be prompted with the following message:  "Are you sure you want to delete this item?"  Click OK to confirm the deletion.

        • Click the Submit Updates button to actually submit the updates to the institution database.

    • Constituent Search

      The Constituent Search menu is a link to the Advanced Search screen where you can do a search for a person and/or company and view their profiles.  

      • Search for a Person

        The Find a person tab allows you to enter the following criteria for a person:  Last Name, First Name, Middle Name, City, State, ZIP Code, Telephone, ID, Major, Class Year, SSN, and Status.  After you enter search criteria and click the Search button, results are displayed if matches are found.

        If multiple matches are found, the list of constituents that match the search criteria is displayed in the Search Results table at the bottom of the screen.  You can then click a constituent name, and the constituent information is displayed in the Relationships Management screen.

        If only one name is found, the Relationship Management screen displays the constituent's information.

        To clear the fields to initiate a new search, click the Reset button.

      • Search for a Company

        The Find a company tab allows you to enter the following criteria for a company:  Name, ID, City, State, ZIP Code, and Telephone.

        After you enter search criteria and click the Search button, results are displayed if matches are found.

        If multiple matches are found, the list of constituents that match the search criteria is displayed in the Search Results table at the bottom of the screen.  You can then click a constituent name, and the constituent information is displayed in the Relationships Management screen.

        If only one name is found, the Relationship Management screen displays the constituent's information.

        To clear the fields to initiate a new search, click the Reset button.

      • View Personal and Company Profiles

        The list of constituents or companies that match the search criteria is displayed.  Major, class year, city, state, and email address may be displayed for a person.  Address, city, state, contact, and email address may be displayed for a company.

        You may be able to click a hyperlinked name to view a personal profile.

         

      • Enter New Information/Edit Existing Information

        While you are viewing constituent information, you may have been given permission to enter new information or edit existing information.  The information you enter/edit is coded according to categories and is available for you to view, edit, and submit to the institution database.  In turn, the Alumni/Development Office personnel will review and process the submissions and update the database.

        Enter New Information

        • If you want to enter new information for a section, click the icon for the appropriate section.

        • Enter the updated information.  The text box allows 2,000 characters.

        • Click the Save or Submit button.

        Edit Existing Information

        • If you want to edit current information for a section, click the appropriate Edit icon  ().

        • Enter the updated information.  The text box allows 2,000 characters.

        • Click the Save or Submit button.

      • Edit/Delete Updates

        If given permission, you can edit/delete updated information before final submission to the institution database for processing.  Current information is displayed at the top of the screen.

        • From the Manage menu, click Submit Updates.

        • A list of updates made is displayed.  The first 50 characters of each update are displayed.

        • Click the Edit icon ( ) to edit or the Delete icon () to delete update(s).

        • After editing, click the Submit Updates button for submission to the institution database.
