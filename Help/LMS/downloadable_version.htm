Downloadable Version

The Downloadable Version portlet might be used by your instructor to post the course syllabus as a handout. If the default template for course sections is being used, the Downloadable Version portlet might be located on a Syllabus page, along with the related Course Syllabus portlet.

The Downloadable Version portlet is an instance of the Handouts portlet. If you are a faculty member or an administrator, and you need help adding content to the portlet, see  e-Racer: Faculty Guide or Jenzabar's Internet Campus Solution: Administration Guide.
