Course Search

The Course Search portlet allows you to search the course catalog.

+ To search for a course

1.        Navigate to an instance of the Course Search portlet.

2.        Enter search criteria in at least one of the following fields:

o      Course Title

o      Course Code

o      Faculty Last Name

o      Term

o      Course Description

3.        If you want the search results to include the description of each course, select the Show Course Descriptions check box.

4.        Click Search.

For each course that meets your search criteria, the portlet will list the following:

·       The course code

·       The name of the course, which also serves as a link to the course context

·       The faculty member, with a blue "i" icon that can be used to display the instructor's My Info pop-up.

·       Meeting times for the course
