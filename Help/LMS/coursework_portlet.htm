Coursework

The Coursework portlet is where you complete assignments, review completed assignments, view assignment grades, and so forth.

There are three different types of assignments that might be represented in the Coursework portlet:

+ Online tests

Some assignments are tests that you complete in the portal. That is, within the portal, you display the test, enter your answers, and submit it for grading. These are known as online tests.

Note that online tests:

·       May be timed. When the end of the time limit is reached, the test will be automatically submitted.

·       May be set up so that if you navigate away from the test, you will be locked out and unable to complete it.

Check with your instructor if you have concerns about rules that have been set up concerning how the test is to be taken.

+ Assignments that you upload

With some assignments, you complete them in another application, such as Microsoft Word, then upload and submit your file through the portal. Depending on the version of e-Racer used by your school, these assignments might be called File Exchange or Uploaded assignments.

+ Other types of assignments

Other types of assignments might be completed in some other way. For example:

·       You might need to make an oral presentation in class.

·       You might need to complete work using a third-party tool that you interact with through the portal.

In both cases, your instructor might want to track your progress using an Offline or Basic assignment (the name will vary depending on the version of e-Racer used by your school).

If you are a faculty member or an administrator, and you need help configuring the portlet or creating assignments, see e-Racer: Faculty Guide.
