For more information

In addition to this help system, which describes the e-Racer portlets, the portal also includes a help system for the base product and its portlets, as well a help system geared toward faculty. Your portal might also include help systems for additional optional modules that your school may have purchased.

If you are looking for details on a portlet that is not described in this help system, try going back to the portlet you need information on and clicking the help icon in the upper right-hand corner, which most portlets should have. This should open the appropriate help pages.

Or you can use the following link:

* Click here to open the help system for base JICS in a new tab

+ To check Jenzabar's support site

1.        Open a browser and navigate to MyJenzabar.net/ics.

2.        Log in to the site.

3.        Choose the Support tab, then use the pane at the left-hand side of the screen to navigate to the appropriate page.
