Sorting items that require approval

If you have numerous items to review and take action on, you may want to sort them.

To sort items that require approval:

  1. In the Item Approval portlet, click the wrench icon. The system displays the Customize Item Approval Portlet screen, with the Preferences tab selected.

  2. Using the Order Criteria drop-down list, specify how you want the items to be sorted.

  3. If appropriate, use the Show Detail drop-down list to specify whether you want to display additional details. For example, if you are reviewing requisitions, the Show Detail feature lets you display the quantity requested along with the price per quantity.

  4. Click Save.
