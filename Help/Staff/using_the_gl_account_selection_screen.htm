Using the GL Account Selection screen

GL Account Selection is a pop-up screen that can be displayed from the Requisition Entry portlet. It lets you choose the account to which your requisition will be charged. The format and behavior of this selection screen varies depending on your privileges. It displays either a list of all accounts to which you have access, or a set of tools that let you search for the right account.

Use one of the following procedures as appropriate:

+ To select an account from a list    

  1. Examine the list and select the account that you want to use. If the list is long, note that you can filter using the Sort By drop-down list.

  2. Click Select Account. The system closes the screen and populates the Enter New Requisition form with the account that you chose.

+ To select an account using the search tools     

  1. Do one of the following, as appropriate: ·) In the Search by Account Number field, enter some or all of the account number. ·) In the Search by Account Description field, enter some or all of the account description. ·) In the Search by Account Component field, enter as much of account segment data as you know. Note that you can use only one of the three preceding methods to search, not a combination of them.

  2. Click Search. The system displays a list of the accounts that meet the criteria you entered. Note that if your search criteria match more than the maximum number that can be displayed, the system will display a dialog stating that these are only partial results and that you may want to try to limit your search further.

  3. Do one of the following: ·) If you see the account that you want to use, select it and click Select Account. ·) If you do not see the account you want, search again using new search terms.
