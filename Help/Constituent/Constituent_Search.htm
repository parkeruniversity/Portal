Constituent Search

NOTE:  In order to utilize the Constituent Search portlet, you must have a login and password.

Constituent Search allows you, if you have a Constituent role, to search for and view contact information related to other constituents and companies.  You can search for a person by name, city, state, major, and class year.  You can search for a company by name.

The My Info and Constituent Search portlets have been specifically designed to accommodate privacy controls, which determine whether you are included in the search results and also determine information that can be viewed by others.  In order for you to be included in the search results and to determine information that can be viewed by other constituents, you must sign into JICS and set your personal privacy controls in the My Info portlet.   

The Constituent Search screen contains the following search fields:

Column

Description

Last Name

If a value is entered here, the search type is PERSON and the comparison is LIKE match.  If a partial name is entered, all names that match the partial name are retrieved.

First Name

If a value is entered here, the search type is PERSON and the comparison is LIKE match.  If a partial name is entered, all names that match the partial name are retrieved.

Middle Name

If a value is entered here, the search type is PERSON and the comparison is LIKE match.  If a partial name is entered, all names that match the partial name are retrieved.

Major

If a value is selected here, the comparison is EXACT match.

Class Year

If a value is entered here, the comparison is EXACT match.

City

If a value is entered here, the comparison is EXACT match.

State

If a value is selected here, the comparison is EXACT match.

(Company) Name

If a value is entered here, the search type is COMPANY and the comparison is LIKE match.  If a partial name is entered, all names that match the partial name are retrieved.

Functions

  • Search for a person

    On the default search screen, Constituent Search criteria are limited to first, middle, and last names of a person.  However, more criteria can be entered if you click the Advanced Search link.  The Advanced Search screen contains a Find for Person tab for entering additional criteria.  When the search is initiated, a list of constituents that match the search criteria is displayed.  Certain other information about the constituents may also be displayed, depending on the constituents' privacy settings in the My Info portlet and data that might be available from the institution's database.

    If you want to clear the fields to initiate a new search, click the Reset button.

  • Search for a company

    On the default search screen and on the Find a Company tab of the Advanced Search screen, Constituent Search criteria are limited to the name of a company.  When the search is initiated for a company, the company profile and a list of employees (constituents) display.   Clicking on an employee name displays their profile information; however, the profile information displayed is individually controlled by each constituent via the privacy controls that are available in the My Info portlet.

    If you want to clear the fields to initiate a new search, click the Reset button.

  • View personal and company profiles

    The list of constituents or companies that match the search criteria is displayed.  Major, class year, city, state, and email address may be displayed for a person.  Address, city, state, contact, and email address may be displayed for a company.

    You may be able to click a hyperlinked name to view a personal profile; however, information displayed in the personal profile is determined by privacy controls set by the Administrator or the individual constituent in the My Info portlet.
