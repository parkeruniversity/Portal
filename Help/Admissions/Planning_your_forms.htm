Planning your forms

Both the Apply Online and Make an Inquiry portlets allow you to create multiple forms for use in that portlet. For example, in the Apply Online portlet, you could create several application forms: one for prospective undergraduates, one for medical school applicants, one for law school applicants, and so forth. Similarly, in the Make an Inquiry portlet, you may want to create two inquiry forms: one for candidates to submit and one for counselors to submit on behalf of candidates.

Once you create forms, they are stored in the system until they are activated. When you activate a form, you publish it to the portlet so that prospective students can begin using it.

Note that with the Make an Inquiry portlet, you can activate only one form at a time. However, if you want to make multiple inquiry forms available simultaneously, you can do so by adding multiple Make an Inquiry portlets to the page. (For details on this process, see “Placing portlets on a page” on page 3)

If you wish, you can set up your forms so that the data candidates enter is written automatically to your database. Doing so eliminates the need for an employee to re-key data.

The following are some terms with which you might want to become familiar before proceeding:

  • Section – A section is a page within a form. Every form must have at least one section, and when you create a second section, you are essentially adding a second page to your form. Each section can be customized with introductory text and images.

  • Header –  A header is simply a heading, which you can use to organize questions in a form.

  • Introduction – Text used to preface questions on the form. Often this text offers instructions to the users filling out the forms.

  • Category – A category is a heading that you can use to group various applications within the Apply Online portlet.
