Browsing for questions

If you want to augment an existing form by adding selected content – but not an entire section – from another form, you should import questions or sections.

To browse for questions within an existing form:

  1. Go to the Browse Forms area of this screen, which contains a list of all the forms that exist in this portlet.

  2. If you want to display forms from a different portlet, use the Browse Forms drop-down list to select a different portlet. The system updates the list of forms.

  3. Click the name of the form that you want to browse. The system displays a screen listing all the sections in that form.

  4. To browse a section, click the name of it. The system displays a screen listing all the sections in that form.

  5. If you want to view the details of any particular question, or see a preview of how it would appear on a form, click the question label. When you’re finished examining these details, click Back to previous screen.

  6. Indicate the questions that you want to import by selecting the checkboxes next to the appropriate question labels.

  7. Click Import Selected.
