Adding a section

In some cases you may want your form to have multiple pages. This may be the case if the form is long, or if you want to create a logical separation between different topics. For example, you might want the first page of a form to cover basic biographical information about the user, and the second page might contain essay questions. In Form Builder, a page is known as a section, so to add a page, you must add a section to your form.

To add a section:

  1. If you do not have a form open for editing, open it, as described in Opening a form for editing.

  2. Click Add a Section.

  3. The system displays a screen with the heading Add a section.

  4. In the Section Name field, enter a name for the first section of your form.

  5. If you want this section to include an image, enter an path in the Image File field.

  6. If you want to include instructions, enter text in the Text field.

  7. If appropriate, use the Image Placement field to specify whether you want the instructions to be displayed to the left of the instructions or below them.

  8. Click Save.

  9. The system saves your changes and again displays the Form Builder: Your Form Name screen.
