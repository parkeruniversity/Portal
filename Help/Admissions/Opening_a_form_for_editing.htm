Opening a form for editing

If you want to make changes to a form that you created previously and saved in your system, first you must open it for editing.

To open a form for editing:

  1. Do one of the following: · Navigate to the Apply Online portlet and click Admin this portlet. · Navigate to the Make an Inquiry portlet and click Admin this portlet. The system displays the Admin screen for the portlet that you chose.

  2. Do one of the following: · Click Manage all applications. · Click Manage all inquiry forms.The system displays a list of all the forms that currently exist in this portlet.

  3. Locate the form that you want to edit. Open it for editing by doing one of the following: · Click the name of the form. · Click the pencil icon that corresponds with the form you want to edit. The system displays a screen that summarizes all the content in this form. From this page you can add sections or add questions, as described in Adding questions.
