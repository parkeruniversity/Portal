Academic Information

The Academic Information portlet displays the following academic information for the candidate:

  • Faculty Advisors

    Names of faculty advisors for the candidate will display on the screen.  If no faculty advisor exists for the candidate, the following message displays:   "No faculty advisor assigned."

  • Intended Majors

    Majors for the candidate will display on the screen.  If no major exists for the candidate, the following message displays:  "A major has not yet been recorded."
