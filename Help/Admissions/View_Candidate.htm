View Candidate

View Candidate allows Admissions Officers and Counselors to search for and view details of their candidates.

Logged-in Counselors can view Candidates assigned to them.  Admissions Officers can view candidates assigned to a Counselor, who, in turn, has been assigned to them.  The candidate name becomes a hyperlink if you have permission to view detailed information for that candidate.  Clicking the name retrieves and displays read-only details from the ERP for the selected candidate.

If you have permission, you can submit an inquiry for a candidate or update existing candidacy information on behalf of a candidate through the Make an Inquiry portlet.  Also, if the EDIT button is available, you can modify existing information and save it to the ERP.  When you click the Submit button, you will be notified if the submission of the candidate's information to the ERP was successful or was not successful.

The View Candidate Details and Return to View Candidate links returns you to the View Candidate screen.

Functions

  • Search for a candidate

    You can search for candidates by entering any or all of the following search criteria:

    ID Number

    Will search for exact match

    Last Name

    Can enter the whole or partial name

    First Name

    Can enter the whole or partial name

    SSN

    Will search for exact match

    Any/All of the Criteria

    If the Any option is selected, will search for names that contain any of the criteria entered; if the All option is selected, will search for names that contain all criteria entered

     

    NOTE for PX:  PX supports the following search criteria:

    • Exact match for ID Number only (no other criteria allowed)

    • Exact match for SSN only (no other criteria allowed)

    • Whole or partial match for Last Name and an optional First Name (no other criteria allowed)

    Clicking the Search button with retrieve and display the candidate and associated candidacy information.

    Clicking the Search button with no search criteria or clicking the View All My Candidates link will retrieve and display all candidates assigned to the Admissions Officer/Counselor.

  • View All My Candidates

    Clicking the View All My Candidates link or the Search button with no search criteria will retrieve and display all candidates assigned to the Admissions Officer/Counselor.

  • View individual candidate

    Clicking on a candidate's name in a retrieved candidate list will display candidate and candidacy information retrieved from the ERP.

  • Update personal information

    Clicking the edit icon () (if available) in the Personal Information section opens the Update Personal Information page and allows you to submit changes to selected personal information.

  • View All Addresses

    Clicking this link opens the View Addresses page, which displays all addresses (including email addresses) for the candidate.  If the edit icon () is available, you can submit address changes.

  • Update Candidates via Inquiry (not available for Jenzabar TE)

    Clicking this link opens the Make an Inquiry portlet.

  • View All Test Scores

    This link opens the View Test Scores page and displays test score data (if available) for the selected candidate.

  • View Attributes

    This link opens the Attributes screen where you can view and submit any changes/additions for the selected candidate.

  • View Contacts

    Clicking this link opens the View Contacts page, where you can view completed and pending (expected/scheduled) contacts between the candidate and your institution.  This page also lists the person responsible for the contact (Action), and clicking the name (if hyperlinked) opens an email form addressed to that person.

  • View Admissions Status

    Clicking this link opens the Admissions Status portlet for this candidate.

  • View Folder Items

    Clicking this link opens the Folder Items portlet for this candidate.

  • View My Candidate List

    Clicking this link displays all candidates assigned to you.

  • Email Counselor/Candidate

    Clicking an email address for a candidate and/or Admissions personnel will open an email form with their email address in the To: line.
