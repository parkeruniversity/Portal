View Activity

View Activity is read only and allows Admissions Officers and Counselors to view statistical and configuration information specific to them and any Admissions personnel who report to them.  Also, they can view detailed information for Candidates assigned to them or Candidates assigned to persons who report to them.

If you are logged in as an Admissions Officer, you can view counts (if more than 0) of Candidates, Territories/Regions, other Admissions Officers, and Counselors assigned to you.  If you are logged in as a Counselor, you can view counts (if more than 0) of Candidates and Territories/Regions assigned to you.  These counts are retrieved from the ERP and only display if the count is more than 0.  If all counts are 0, a message alerts you to this fact and no counts are displayed.

Preferences

  • Hide Server Data

    This setting allows you to hide data returned from the ERP to reduce display time of the default screen only.

    • From the View Activity portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Select the check box.

    • Save.

Functions

  • View Detail

    The View Detail link may be available.  This link accesses a page that displays details for the selected year and term for the Admissions Officer or Counselor who is logged in (default).  It displays Candidate counts by Stage and a list of assigned territories/regions if this information is available from the ERP.  If the user has the role of Admissions Officer, also displayed is a list of Admissions Officers and/or Counselors assigned to them.  Although this page displays information for the logged-in user, activity details for a selected Admissions Officer and/or Counselor can be displayed.

  • View Candidates

    This link takes you to the View Candidate portlet.
