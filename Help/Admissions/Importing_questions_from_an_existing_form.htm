Importing questions from an existing form

If you want to augment an existing form by adding selected content – but not an entire section – from another form, you should import questions or sections.

To import sections from an existing form:

  1. If the form is not already open for editing, open it, as described in Opening a form for editing.

  2. Click Import Question(s). The system displays a screen with the heading Import Questions.

  3. Do one of the following: · Search for the question without opening another form · Browse an existing form to select questions

  4. If appropriate, click “Preview this question” to see how the question will be displayed. Make any changes that are  necessary.

  5. Click Save.
