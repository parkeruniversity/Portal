Admissions Status

Admissions Status allows you to view information about and status of active current and future candidacies.  Related program information displayed can include entry term, entry classification, majors, education goals, location, admission status, contacts, and a suggested next step.  If you are a Candidate, you may be able to contact your Admissions Counselor through an email link.

Functions

  • View Admissions Status

    • If you are an Admissions Administrator, Admissions Officer, or Admissions Counselor, clicking the View Admissions Status link opens the Find Candidate screen where you can search for specific candidates by ID Number.  When an ID Number is entered and a match is found, the Admissions Status screen opens.

    • If you are a Candidate, clicking the View Admissions Status link opens the Admissions Status screen.

    • The Admissions Status screen displays the most recent candidacy for the ID Number.  It also displays related information about that candidacy from the ERP.

    • If  multiple candidacies are found for the ID Number, the additional candidacies can be selected and viewed from the Program drop down list.
