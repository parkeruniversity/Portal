Folder Items

Folder Items portlet displays the following:

  • Candidacy Program

    • The most current candidacy is displayed.

    • If multiple candidacies are found for the ID Number, the additional candidacies are available from the Program drop-down list.

  • Requirements Not Yet Met

    • Items received by the Admissions Office but not meeting the required minimum are designated by a green check mark ().

    • Missing items the Admissions Office requires to process the candidate's admission application are designated by a red x (x).

  • Requirements Met

    • Items meeting the required minimum received by the Admissions Office are listed along with the date received.

Functions

  • View Folder Items

    • If you are an Admissions Administrator, Admissions Officer, or Admissions Counselor, clicking the View Folder Items link opens the Find Candidate screen where you can search for specific candidates by ID Number.  When an ID Number is entered and a match is found, the View Folder Items screen opens.  Also available on the View Folder Items screen is a Find Candidate link that returns you to the Find Candidate screen where you can enter another Candidate ID Number.

    • If you are a Candidate, clicking the View Folder Items link opens the View Folder Items screen.

    • The View Folder Items screen displays the most current candidacy and associated folder items for the ID Number.

    • If multiple candidacies are found for the ID Number, the additional candidacies can be selected and viewed from the Program drop-down list.

    • Both items received and missing are displayed.  Also displayed is the date of items received.
