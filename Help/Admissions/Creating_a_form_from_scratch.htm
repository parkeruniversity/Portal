Creating a form from scratch

If you don’t have any existing forms in your system, or if you want to create a form with questions that don’t exist in your other forms, create your form from scratch.

To create a form from scratch:

  1. Do one of the following: · Navigate to the Apply Online portlet and click Admin this portlet. · Navigate to the Make an Inquiry portlet and click Admin this portlet. The system displays the Admin screen for the portlet that you chose.

  2. Do one of the following: · If you working in the Apply Online portlet, click Add A New Application. · If you’re working in the Make an Inquiry portlet, click Add a new form.

  3. The system displays the Form Builder: New Form screen.

  4. In the Form Name field, enter a name for the form.

  5. Click Add New Content.

  6. The system displays the Form Builder: Your Form Name/Add a Section screen.

  7. In the Section Name field, enter a name for the first section of your form. Each form must have at least one section. A section is essentially a page within the form.

  8. If you want this section to include an image, enter an path in the Image File field.

  9. If you want to include instructions, enter text in the Text field.

  10. If appropriate, use the Image Placement field to specify whether you want the instructions to be displayed to the left of the instructions or below them.

  11. Click Save. The system displays the Form Builder: Your Form Name screen, which lets you add questions and make other modifications to your form.

  12. Add as many questions as are necessary, as described in “Adding questions” on page 3.

  13. If appropriate, add additional sections (pages) to your form

  14. Click Save.
