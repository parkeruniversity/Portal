Activating Inquiry forms

Note that the Make an Inquiry portlet only allows one form to be activated at any given time. If you want multiple Inquiry forms to be available simultaneously, add additional Make an Inquiry portlets to the site.

To activate an inquiry form:

  1. Navigate to the Apply Online portlet and click Admin this portlet. The system displays the Admin screen for the portlet.

  2. Click Activate an inquiry form. The system displays a list of all the inquiry forms.

  3. Select the radio button next to the form you want to activate. Click Choose form.
