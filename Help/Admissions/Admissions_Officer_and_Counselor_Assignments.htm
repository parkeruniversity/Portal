Admissions Officer and Counselor Assignments

Admissions Officer and Counselor Assignments allows only the Admissions Administrator to assign Counselors and Admissions Officers to other Admission Officers.  This portlet can be used when a vice-president oversees multiple admissions directors and needs to have the directors' information in addition to information related to the counselors that the directors oversee.  Once this global setting is defined via the Site Manager link, these assignments can then be used to filter data so that Admissions Officers can view information for Admissions Officers and Counselors assigned to them as well as Candidates assigned to those Admissions Officers and Counselors.  This information is viewed in the View Candidate and View Activity portlets.

Functions

  • Assign Admissions Officers/Counselors

    • From the Admissions Officer drop-down list on the default screen, select the person to which you want to view or change assignments.  The list contains all persons with the role(s) specified by the Is Displayed in ADO Dropdown operation defined by the JICS Administrator.  The names are listed alphabetically by first name, and there is no name selected by default.

    • All Admissions Officers/Counselors available for assignment to the selected person are displayed in the Available Admissions Officers & Counselors list.  Check boxes are selected for Admissions Officers/Counselors already assigned to the selected Admissions Officer.

    • You can assign all persons, assign selected persons, or remove selected persons by selecting the appropriate check boxes.

    • Save.

    • If you wish, you can select another Admissions Officer to view or change assignments.
