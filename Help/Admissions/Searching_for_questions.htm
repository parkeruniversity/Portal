Searching for questions

If you've begun the process of importing content from a form, as described in Importing questions from an existing form, and you want to search for a specific question, use this procedure.

To search for a question:

  1. In the Search for a Question area of the screen, enter as much data as is appropriate.

  2. Click Search. The system returns a list of all the questions that meet the criteria you entered.

  3. If you want to view the details of any particular question, or see a preview of how it would appear on a form, click the question label. When youâ€™re finished examining these details, click Back to previous screen.

  4. Indicate the questions that you want to import by selecting the checkboxes next to the appropriate question labels.

  5. Click Import Selected.
