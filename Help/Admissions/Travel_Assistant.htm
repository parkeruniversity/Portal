Travel Assistant

Travel Assistant displays travel-related links to Web sites and internal pages that can be used by the Admissions Officers and Counselors to access helpful travel information.

By default, only Admissions Administrators can edit and manage Travel Assistant links by activating and inactivating links, repositioning the links, adding links and link headers, modifying links, deleting links, and defining hyperlinked text.  However, you may be able to manage Travel Assistant links depending on the security you have been given.  For more information about these operations, see the Bookmarks topic in Getting Started with JICS.

The following three links are available by default:

  • MapQuest (http://www.mapquest.com)

  • Travelocity (http://www.travelocity.com)

  • The Weather Channel (http://www.weather.com)

Preferences

The following preferences may be available; however, depending on security given to you, indicated results may not occur if you select any of the preference check boxes.

  • Order Bookmarks Alphabetically

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Preferences tab.

    • Select this check box to automatically order the bookmarks alphabetically.  Clear this check box if you want to manually order the bookmarks.

  • Order Sets Alphabetically

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Preferences tab.

    • Select this check box to automatically order sets alphabetically.  Clear this check box if you want to manually order the sets.

  • Show Empty Sets

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Preferences tab.

    • Select this check box to display sets that have no bookmarks.

  • Show Editing Tools by Default

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Preferences tab.

    • Select this check box to show editing tools on the Main screen by default.  Clear this check box to hide editing tools on the Main screen.  These editing tools include check boxes, form elements, and editing icons.

  • Log Bookmark Clicks

    • Select this check box to automatically order the bookmarks alphabetically.  Clear this check box if you want to manually order the bookmarks.

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Preferences tab.

    • Select this check box to have the system keep a log of each time a bookmark is clicked and display this information on the Main and Reporting screens.  Clear this check box if you do not wish to have the system keep and display a log of bookmark clicks.

  • Use Display Indicator

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Preferences tab.

    • Select this check box to use the display indicator to set when and how long a new bookmark will be active.  Clear this check box to allow bookmarks to be displayed immediately and indefinitely.
