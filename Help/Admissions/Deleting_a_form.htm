Deleting a form

If a form is no longer needed, you can delete it.

To delete a form:

  1. Do one of the following: · Navigate to the Apply Online portlet and click Admin this portlet. · Navigate to the Make an Inquiry portlet and click Admin this portlet.The system displays the Admin screen for the portlet that you chose.

  2. Do one of the following: · Click Manage all applications. · Click Manage all inquiry forms.The system displays a list of all the forms that currently exist in this portlet.

  3. Locate the name of the form that you want to delete, and click the corresponding trash can icon
