Setup Steps for QX

In order to successfully implement Admissions Officer, the following steps should be performed in the order in which they are listed.

NOTE:  Persons who apply for a program in an academic term that ends later than today are considered to be Candidates and may not have counselors assigned to them.  The counselor assigned to the particular program can view those Candidates who applied to that program.

NOTE:  Your JICS Administrator has already defined roles that have access to individual portlets via the Allow Access checkbox (Site Manager link, Global Portlet Operations Edit link).  For more information about the Allow Access global operation, see Getting Started with JICS.

Step 1:  In QX, create JICS accounts for the Admissions Administrator.

  • Select the individuals that have Admissions Administrator privileges and add that web group to their ID Number in CampusWeb Admin.

    • Navigate to the User Data window (Options, User Data).

    • Using the search, select the user that is to become the Admissions Administrator.

    • In the Web Access tab, use the right mouse button to bring up the popup menu and select Add Group.  Choose Admissions Administrator from the list.

    • Make any other changes necessary and save your changes.

  • Provide these individuals with their login information.

Step 2:  In QX, create JICS accounts for Admissions Officers (not required).

  • Select the individuals who get Admissions Officer privileges and add that web group to their ID Number in CampusWeb Admin.

    • Navigate to the User Data window (Options, User Data).

    • Using the search, select the user that is to become the Admissions Officer.

    • In the Web Access tab, use the right mouse button to bring up the popup menu and select Add Group.  Choose Admissions Administrator from the list.

    • Make any other changes necessary and save your changes.

  • Provide these individuals with their login information.

Note:  You are not required to assign anyone the role of Admissions Officer.  JICS has provided two groups of Admissions employees so different permissions may be set based on these groups.

Step 3:  In QX, create JICS accounts for Admissions Counselors.

  • Any constituent who is a counselor in QX Admissions will be an Admissions Counselor when their JICS account is created using CampusWeb Admin.

    • Navigate to the User Data window (Options, User Data).

    • Using the search, select the user that is to become an Admissions Counselor.

    • Verify that Admissions Counselor is displayed in the Const Roles list.

    • Make any other changes necessary and save your changes.

  • Provide these individuals with their login information.

Step 4:  In JICS, set security permissions by role for each portlet.

  • Account Notification

    • From the Account Notification portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Security tab.

    • Select the appropriate Operations check boxes for each role or select the All Operations check box.

      • Cancel Notification

        Selecting this check box for a role allows persons with that role to cancel an account notification for a JICS user via the Held Notifications tab on the View New Accounts screen.

      • Send Email

        Selecting this check box for a role allows persons with that role to send an email with login and password information to a JICS user via the View New Accounts screen.

      • Can Set Up Portlet

        Selecting this check box for a role allows persons with that role to see the Setup link in the Account Notification toolbar.

      • Create Email Template

        Selecting this check box for a role allows persons with that role to create a new email template for use in sending an email with login and password information to a JICS user via the View New Accounts screen.  When this permission is given, the add icon () is available on the Setup screen.

      • Modify Email Template

        Selecting this check box for a role allows persons with that role to edit an existing email template for use in sending an email with login and password information to a JICS user via the View New Accounts screen.  When this permission is given, the edit icon () is available on the Setup screen.

      • Move to Active

        Selecting this check box for a role allows persons with that role to move an account notification from Held to Active status on the View New Accounts screen.

      • Move to Held

        Selecting this check box for a role allows persons with that role to move an account notification from Active to Held status on the View New Accounts screen.

    • Save.

    • If you need to add a role, click the Add a Role link.  See Getting Started with JICS for more information about adding a role.

    • Exit.

  • Admissions Status

    • From the Admissions Status portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Security tab.

    • Select the appropriate Operations check boxes for each role or select the All Operations check box.

      • Can Edit Stage Setup

        Selecting this check box for a role allows persons with that role to have access to the Stage Setup screen and to update the stage setup.

      • Can Find Candidates

        Selecting this check box for a role allows persons with that role to search for and open Admissions Status information about a candidate.  Otherwise, they will only be able to see information about themselves.

    • Save.

    • If you need to add a role, click the Add a Role link.  See Getting Started with JICS for more information about adding a role.

    • Exit.

  • Folder Items

    • From the Folder Items portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Security tab.

    • Select the appropriate Operations check boxes for each role or select the All Operations check box.

      • Can Find Candidates

        Selecting this check box for a role allows persons with that role to search and open Folder Items information about a candidate.

    • Save.

    • If you need to add a role, click the Add a Role link.  See Getting Started with JICS for more information about adding a role.

    • Exit.

  • Travel Assistant

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Security tab.

    • Select the appropriate Operations check boxes for each role or select the All Operations check box.

      • Can Administer Portlet

        Selecting this check box for a role allows persons with that role to have access to all operations for this portlet.

      • Can Manage Bookmarks

        Selecting this check box for a role allows persons with that role to add, edit, and delete helpful links.

      • Can Manage Sets

        Selecting this check box for a role allows persons with that role to add, edit, delete, and order sets.

    • Save.

    • If you need to add a role, click the Add a Role link.  See Getting Started with JICS for more information about adding a role.

    • Exit.

  • Initiate Contact

    • From the Initiate Contact portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Security tab.

    • Select the appropriate Operations check boxes for each role or select the All Operations check box.

      • Can Administer Initiate Contact

        Selecting this check box for a role allows persons with that role to define the text displayed on the Request a Contact and Schedule a Visit screens.

    • Save.

    • If you need to add a role, click the Add a Role link.  See Getting Started with JICS for more information about adding a role.

    • Exit.

  • View Activity

    • From the View Activity portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Security tab.

    • Select the appropriate Operations check boxes for each role or select the All Operations check box.

      • Assigned Groups to Display

        Selecting this check box for a security group will display that group's assigned subordinates.

    • Save.

    • If you need to add a role, click the Add a Role link.  See Getting Started with JICS for more information about adding a role.

    • Exit.

  • Make an Inquiry

    • From the Make an Inquiry portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Security tab.

    • Select the appropriate Operations check boxes for each role or select the All Operations check box.

      • Can Create a Form

        Selecting this check box for a role allows persons with that role to create a form for this portlet.

      • Can Delete a Form

        Selecting this check box for a role allows persons with that role to delete a form from this portlet.

      • Can Modify a Form

        Selecting this check box for a role allows persons with that role to modify a form for this portlet.

      • Can Setup Portlet

        Selecting this check box for a role allows persons with that role to have access to the Setup menu for the Make an Inquiry portlet.

    • Save.

    • If you need to add a role, click the Add a Role link.  See Getting Started with JICS for more information about adding a role.

    • Exit.

Step 5:  In JICS, assign Admissions Administrators responsible for account notification.

  • Request the JICS Administrator to assign the Admissions Administrators who will be responsible for sending account notifications to Admissions Officers and Counselors.

Step 6:  In JICS, create Division and Major/Program combinations and assign to Admissions personnel.

  • From the Division and Major Assignment portlet, click the Site Manager link in the upper right corner of the screen.

  • Click the Edit link for Global Portlet Operations.

  • Scroll down to locate Global Operations for Division and Major Assignment.

  • Select the appropriate roles for the following operation:

    • Is Displayed in User Dropdown?

      Users in the selected groups will be available in the Admissions Personnel drop down on the Division/Major and User Assignment screen when you click the Assign admissions personnel to division/major combinations link from the portlet.

  • Save.

  • Then create combinations and assignments using the Division and Major Assignment portlet.

    • Click the Create the division and major/program combinations link.

      • On the Division and Major/Program Assignment screen, select the division you want to work with from the Division drop-down list.  The divisions are listed alphabetically, and there is no division selected by default.

      • Select or clear the check box(es) beside the major/program you want to add to the selected division.

      • Save.

    • Click the Assign admissions personnel to division/major combinations link.

      • On the Division/Major and User Assignment screen, select the appropriate person you want to work with from the Admissions Personnel drop-down list.  The list contains all personnel specified by the Is Displayed in User Dropdown global operation defined by the JICS Administrator for this portlet.  The personnel are listed alphabetically by first name, and there is no person selected by default.

      • Select or clear the check box(es) beside the major/program you want to assign to the selected person.

      • Save.

Step 7:  In JICS, configure the Account Notification portlet.

  • From the Account Notification portlet, click the Site Manager link in the upper right corner of the screen.

  • Click the Edit link for Global Portlet Operations.

  • Scroll down to locate Global Operations for Account Notification.

  • Select the appropriate roles for the following operation:

    • Can Receive Account Notification?

      Users in the selected groups will receive account notifications.

  • Save.

  • Exit.

  • Only ONE of the Admissions personnel who have been assigned the account notification responsibility will need to perform the following configuration of the Account Notification portlet.

    • Click the Setup is required before account notification will perform properly link on the Main screen, which opens the Setup screen.  Enter the following configuration information:

      • From the Security Groups to Send Account Notifications to drop-down list, select the appropriate security group to which to send account notifications.   Security groups in this list have been given the Can Receive Account Notification global operation and have not already been assigned to another instance of the portlet.

      • Select the Default Email Template for Account Notifications.  Displayed are the email templates available for the security group selected above in the Security Groups to Send Account Notifications to list.

        You can also create new or edit existing email templates that will be used by the Account Notification portlet.  These templates should include the cc, bcc, subject, and body of the email and can be customized to include the user's name, major, login, and password.  The options to create or modify email templates are available if Modify Email Template and Create Email Template check boxes were selected as operations for the security group.  For more information, see the Email Templates topic in Getting Started with JICS.

      • In Account Notification Delivery Method, decide if candidates will be notified by automatic email or manual email that an account has been created for them.

        • If Automatic Email Notification is selected, an email will be automatically sent without any intervention required by the Admissions Administrator.  This option is not recommended for the initial batch account notification.

        • If Manual Email Notification is selected, a temporary table in JICS will collect the new accounts and the Admissions Administrator will review and send emails through the Account Notification portlet.  This option is recommended for the initial batch account notification.

Note: For more information on the Account Notification portlet, see Getting Started with JICS.

Step 8:  In JICS, configure the Initiate Contact text.

You can configure the text that displays at the top of the Request a Contact and Schedule a Visit screens.  This text can contain any instructions and information useful in submitting the request.

To configure this text:

  • Click the Setup menu in the Initiate Contact portlet to open the Configure Initiate Contact Text screen.

  • You can retain the default text, edit the default text, or enter customized text for your institution (maximum is 3000 characters).

  • Save.

 

Step 9:  In JICS, configure the Admissions Status portlet.

Note:  Only ONE Admissions Administrator should perform this step.

  • Order and/or hide the following candidacy items as to how you would like them to display in the Field of Study section:  Location, Expected Incoming Classification, Education Goal, Entry Term, Major, Secondary Major.

    • Click the wrench icon in the Admissions Status portlet.

    • Click the Settings tab.

    • For each item, select its position (1-6) from the drop-down list.  You can select Hide if you do not want that item to display.

    • Click Save.

    • Click Exit.

  • Edit stage setup for QX.

    • From the Admissions Status portlet, click the Setup menu option or the Setup link.

    • The stages from QX are displayed on the Stage Configuration screen.  If a new stage is added in QX, the stage and its description will automatically display on this screen.

    • Configure the following for each stage:

      • Portlet Stage

        Select this check box if you want this stage to display in the Status section of the Admissions Status portlet.

      • Portlet Stage Message

        Enter a message that will appear in the Admissions Status portlet if the candidate's current QX stage is a JICS stage (i.e., Portlet Stage check box is selected for this stage).

      • Non Portlet Stage Message

        Enter a message that will appear in the Admissions Status portlet if the candidate's current QX stage is not a JICS stage (i.e., Portlet Stage check box is not selected for this stage).  You may want to display a custom generic message to candidates rather than stages like Held or Defer.

      • Show Apply Link?

        Select this check box if you want the apply link to display in the Your Next Step section of the Admissions Status portlet.  This check box should be selected for only those stages that are pre-application type stages.

    • Save.

 

Step 10:  In JICS, configure the Make an Inquiry portlet.

When the Candidate or Admissions Officer/Counselor completes the information and clicks Submit, the inquiry form is sent to a holding table in QX until an Admissions Administrator chooses to view or print the form.  The information can then be entered into the respective Admissions tables.  At that time, a valid ID Number can be created for the Candidate in QX.

  • Select the form type to be configured for the portlet.

    • From the Make an Inquiry portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Settings tab.

    • Select the appropriate form type.

    • Save.

    • Exit.

  • From the Make an Inquiry portlet, select the Setup menu option to open the Work with Forms screen.  You can create a new form, select a form to be modified, and select a form to be deleted.

  • For more information on creating and configuring forms, please see the Forms and the Using Tags in Email topics.

 

Step 11:  In JICS, build a Community Connections page.

  • The Community Connections portlet no longer exists in JICS as of the 6.0 release; however, if you wish to continue having the Community Connections functionality, Jenzabar recommends that you create a Community Connections page and add the following portlets to that page:

    • Initiate a Contact

    • Helpful Documents (Base Handouts portlet)

    • Community Connection Links (Base Bookmarks portlet)

    • Chat (Base Chat portlet)

    • School Directory (Base Directory portlet)

  • For more information on building pages, see Getting Started with JICS.

Step 12:  In JICS, assign Counselors and other Admissions Officers to Admissions Officers.

  • From the Admissions Officer and Counselor Assignments portlet, click the Site Manager link in the upper right corner of the screen.

  • Click the Edit link for Global Portlet Operations.

  • Scroll down to locate Global Operations for Admissions Officer and Counselor Assignments.

  • Select the appropriate roles for the following operations:

    • Is Displayed in ADO Dropdown?

      Users in the selected groups will be available in the dropdown to which to assign other Admissions personnel.

    • Is Displayed in Selection List?

      Users in the selected groups will be available for assignment to other Admissions personnel.

  • Save.

  • Exit.

  • The Admissions Administrator determines the Admissions Officer who will be responsible for each Counselor and other Admissions Officers and makes these assignments from the Admissions Officer/Counselor Assignment portlet.

    • From the Admissions Officer drop-down list on the default screen, select the person to which you want to view or change assignments.  The list contains all persons with the role(s) specified by the Is Displayed in ADO Dropdown operation defined by the JICS Administrator.  The names are listed alphabetically by first name, and there is no name selected by default.

    • All Admissions Officers/Counselors available for assignment to the selected person are displayed in the Available Admissions Officers & Counselors list.  Check boxes are selected for Admissions Officers/Counselors already assigned to the selected Admissions Officer.

    • You can assign all persons, assign selected persons, or remove selected persons by clearing the appropriate check boxes.

    • Save.

    • If you wish, you can select another Admissions Officer to which to view or change assignments.

Step 13:  In JICS, customize the Travel Assistant portlet.

The functionality of the Travel Assistant portlet is the same as the Bookmarks portlet.

  • Log Bookmark Clicks

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Settings tab.

    • Select this check box to have the system keep a log of each time a bookmark is clicked and display this information on the Main and Reporting screens.  Clear this check box if you do not wish to have the system keep and display a log of bookmark clicks.

  • Use Display Indicator

    • From the Travel Assistant portlet, click the wrench icon to open the Portlet Settings and Preferences screen.

    • Click the Settings tab.

    • Select this check box to use the display indicator to set when and how long a new bookmark will be active.  Clear this check box to allow bookmarks to be displayed immediately and indefinitely.

  • Show/Hide Editing Tools

    • From the Travel Assistant portlet, select Show Editing Tools or Hide Editing Tools from the Manage menu.

    • If Show Editing Tools is the option, all check boxes, form elements, and editing icons will be displayed on the Main screen.  If Hide Editing Tools is the option, no tools will be displayed on the main screen.

    • Click Save.

  • Add a Bookmark

    • From the Travel Assistant portlet, select Add a Bookmark from the Manage menu to open the Manage Bookmark View screen.

    • Enter appropriate information for the new link.

    • Click Save or Save and Add Another.

  • Add a Set

    • From the Travel Assistant portlet, click the Add a Set option from the Manage menu to open the Manage Set View screen.

    • Enter the name of the new set and a description (if desired) and select the location.

    • Click Save or Save and Add Another.

  • Edit a Set

    • From the Manage menu of the Travel Assistant portlet, click Edit a Set and then select the appropriate set to open the Manage Set View screen for that set.

    • Edit desired information.  You can also add a bookmark and add a set.

    • Save.

  • Order Sets

    • From the Travel Assistant portlet, click the Order Sets option from the Manage menu to open the Order Sets View screen.

    • Enter the desired order of the sets.  You can also add a bookmark, edit a set, and add a set.

    • Save.
