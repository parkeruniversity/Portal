Using Tags in Email

Tags are used to insert personal candidate information into emails you create.  If you use a tag in an email template, be sure it is also an assigned question on the inquiry or application you created.  

The following example shows the use of tag insertion:

  • From the Email Template screen, compose your email in the Email Message text box.

  • If you desire to use any of the available tags, type @@ and the appropriate tag in the location you want it to appear in the email.  Tag names for questions can be viewed in the XML Tag Name field on the Edit Test Box Question screen (Navigation:  Make an Inquiry or Apply for Admissions portlets, Setup, Assign questions and headers to form, Select questions and headers, select Edit icon for question).

  • Jenzabar suggests that you send a test email to verify that the tags are valid.  Upon sending the test email, the tag description will replace the @@tag if the tag is valid.  If the tab is invalid, the @@tag will still display.

  • When emails are sent, the system retrieves the information from the user's response to the question and replaces the tag with this data.
