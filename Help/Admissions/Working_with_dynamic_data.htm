Working with dynamic data

Setting up e-mail templates from scrThe Apply Online and Make an Inquiry portlets let you create e-mail templates which are sent to candidates in certain situations. This section details how you can configure these e-mails to meet your business needs.

If you want your automated e-mails to use data about the candidate – such as a personal greeting that uses the candidate’s name – you can do so by inserting tags into your e-mail templates. Then, when e-mails are sent, the system retrieves the information from the user's response to the question and replaces the tag with this data.

Before using a tag in an e-mail template, make sure that it is also an assigned question on a form that you created.

To see a list of all tags assigned to questions:

  1. In either the Apply Online and Make an Inquiry portlet, as appropriate, click Setup.

  2. Click Assign questions and headers to form.

  3. Click Select questions and headers.

  4. Click the Edit icon for the question.

  5. The tag names are displayed in the XML Tag name field on the Edit Text Box Question screen.

To use tags in e-mail templates:

  1. Display the text box for the e-mail template with which you want to work.

  2. Type the characters @@, then the appropriate tag.

  3. Send a test message to make sure that the data is displayed correctly and to ensure that you’re not using an invalid tag.
