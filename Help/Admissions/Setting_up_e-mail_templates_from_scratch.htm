Setting up e-mail templates from scratch

If you want to create new templates for automated e-mails, use this procedure.

To set up e-mail templates from scratch:

  1. If the form is not already open for editing, open it, as described in “Opening a form for editing” on page 32.

  2. Click the “Set up completion messages” link. The system displays a screen with the heading Completion Messages. This screen contains two main sections: one labeled Successful Submission and Unsuccessful Submission.

  3. If appropriate, use the on-screen message field to enter a dialog to be displayed on the user’s screen after he or she successfully submit this form.

  4. By default, the successful submission e-mail will be sent after every successful submission. However, you can change this. Use the “After a successful submission:” drop-down list to choose Ask the User or Never Send an Email, as appropriate.

  5. In the area of the screen labeled Message, enter: · In the From field, a valid e-mail address. • If appropriate, valid e-mail addresses in the CC or BCC fields. · In the Subject field, text to be displayed in the subject line of the e-mail message sent to users. · In the Message field, the message to be displayed in the body of the e-mail notification. (Note: For instructions on how to add dynamic data to the e-mail message, see .)

  6. Scroll down to the area of the screen labeled Unsuccessful Message.

  7. If appropriate, use the on-screen message field to enter a dialog to be displayed on the user’s screen after he or she fails to successfully submit a message.

  8. By default, the unsuccessful submission e-mail will be sent after failed submission. However, you can change this. Use the “After a successful submission:” drop-down list to choose Ask the User or Never Send an Email, as appropriate.

  9. In the area of the screen labeled Message, enter: · In the From field, a valid e-mail address. · If appropriate, valid e-mail addresses in the CC or BCC fields. · In the Subject field, text to be displayed in the subject line of the e-mail message sent to users. · In the Message field, the message to be displayed in the body of the e-mail notification. (Note: For instructions on how to add dynamic data to the e-mail message, see .)

  10. Click Save.
