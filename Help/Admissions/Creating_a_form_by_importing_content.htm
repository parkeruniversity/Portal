Creating a form by importing content

If you  want to create a form that uses one or more sections from another form in your system, you can create a new form by importing data from an existing one.

To create a form by importing content:

  1. Do one of the following: · Navigate to the Apply Online portlet and click Admin this portlet. · Navigate to the Make an Inquiry portlet and click Admin this portlet. The system displays the Admin screen for the portlet that you chose.

  2. Do one of the following: · If you are working in the Apply Online portlet, click Add A New Application. · If you’re working in the Make an Inquiry portlet, click Add a new form. The system displays the Form Builder: New Form screen.

  3. In the Form Name field, enter a name for the form.

  4. Click Import Existing Content. The system displays a screen with the heading Import Sections. It lists all the forms that exist in the portlet that you are currently working in.

  5. If you want to import data from a portlet other than the one you are working in, use the Browse Forms drop-down list to choose a different portlet. The system updates this screen to list the forms that exist in the portlet that you specified.

  6. Locate the form from which you want to import questions, and do one of the following: · To import the entire form, click the Import link, which is in the column at the far right. · To import a section of a form: a Click the name of the form. The system displays a screen that lists all the sections in the form. b Use the checkboxes at the left to select the sections that you want to import. c Click Import Selected. The system displays a screen summarizing all the data that you’ve imported.

  7. If appropriate, click “Preview this question” to see how the question will be displayed. Make any changes that are  necessary.

  8. If you want to add additional sections or questions, you can do so now. For details, see “Adding questions” on page 33 or “Adding a section to your form” on page 35.

  9. Once you are satisfied with the form, click Save.
