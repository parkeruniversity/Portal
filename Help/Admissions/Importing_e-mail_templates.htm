Importing e-mail templates

If you want your form to use the same e-mail templates as another form, you can import templates.

To import e-mail templates:

  1. If the form is not already open for editing, open it, as described in Opening a form for editing.

  2. Click the “Set up completion messages” link. The system displays a screen with the heading Completion Messages.

  3. Click “Import the ‘Completion Messages’ from another form” link. The system displays a screen with the heading Import Settings. This screen lists every form from which you could import e-mail templates.

  4. Use the checkboxes at the left to select the form whose e-mail templates you want to import.

  5. Click Import Selected.
