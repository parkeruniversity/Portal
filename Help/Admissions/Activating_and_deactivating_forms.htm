Activating and deactivating forms

You can activate as many applications at a single time as is appropriate.

To activate or deactivate an application:

  1. Navigate to the Apply Online portlet and click Admin this portlet. The system displays the Admin screen for the portlet.

  2. Locate the form  you want to activate. Click the pencil icon thatâ€™s displayed to the right of the form.

  3. The system displays the Activate an application screen. In the area of the screen labeled Active from:, do one of the following: · To make the application active now, select the Display now radio button. · To deactivate the application, select the Display later manually radio button.: · To make the application active at a specific date in the future, select the Display on field, and enter a date in the corresponding text field.

  4. If the area of the screen labeled Active until: is active, do one of the following: : · To make the application active indefinitely, select the No end date radio button. : · To deactivate the application, select End now. Note that doing so grays out the choices in the Active from: area of the screen. : ·To deactivate the application on a specific date in the future, select the End on radio button, and enter a date in the corresponding text field.

  5. Click Save.
