Adding questions

Form Builder lets you create and add several types of questions to your forms. The process is similar regardless of what type of question you are adding.

Note: You can also import questions from other forms. For details on this process, see Importing questions from an existing form.

To add a question:

  1. If the form is not already open for editing, open it, as described in “Opening a form for editing” on page 32.

  2. Click Add a new Question. The system displays the Add a Question page.

  3. Select one of the four options displayed:  ·Text field question type · Essay question type · Drop-down Menu question type · Multiple Choice question type (when you choose this type, users respond by selecting either checkboxes or radio buttons).

  4. The system displays a page that lets you design your question.

  5. In the Question Label field, enter a label for the question. This label typically is not displayed to users, but when you are working within Form Builder to arrange questions and so forth, this is the label that identifies the question.

  6. In the Question Text field, enter the question or label that the user will see. You can use the  formatting tools at the top edge of the text box to change the font of your text, add emoticons, and so forth.

  7. If this a Text field question, and if you want the user’s answer to adhere to a specific format already described in your system, you can use Validate drop-down list to configure the form to make sure that the format is correct. For example, if you want the user to enter a date, you can use the Validate drop-down list to make sure that the date is entered in the correct format.

  8. In the Required field, indicate whether you want this field to be required.

  9. If this is an Essay question, skip ahead to step 14.

  10. If this is a Multiple Choice question, use the Response Type field to indicate whether the user can select more than one option (via checkboxes) or to be restricted to one choice (via radio buttons).

  11. In the Prepopulate field, indicate whether you want the form to be prepopulated with the data about the user that may already exist in the system.

  12. In the Orientation field, indicate whether you want the question to be displayed to the left of the field where the user inputs his or her answer, or whether you want the question displayed above.

  13. If this is a Text or Drop-down menu type question, skip ahead to step 14.

  14. Do one of the following: · If the answer to this question is one provided by the user, select Static in the Option Type field. ·• If the answer to this question is one that is determined programmatically.

  15. Do one of the following: · In the Static Options area of the screen, enter the choices from which you want the user to be able to select. · In the field at the bottom of the screen, enter the SQL query that will determine the answer to the question.

  16. If appropriate, click “Preview this question” to see how the question will be displayed. Make any changes that are  necessary.

  17. Once you are satisfied with the form, click Save.
